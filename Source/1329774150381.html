Loading or importing the Patient Anatomy

  • In the compare window, move the mouse cursor over the RTView tab.
  • The RTView pop-up window will be displayed:
RTView pop-up window

The RTView pop-up window contains the following buttons:

Load patient anatomy
Import patient anatomy from the DICOM network server
Close patient anatomy
Enlarge the window to the size of the program window
In the enlarged view, all representation options are available.

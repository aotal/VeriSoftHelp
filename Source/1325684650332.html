Importing Matrices from a DICOM Network Server

  • Select the DICOM SCP import mode in the DICOM Import dialog.
  • The DICOM server set via menu Tools → Options on the DICOM tab is displayed.
  • Find the desired data as described in section Finding a Matrix on a DICOM Network Server.
  • Import the data as described in section Selecting and Importing a Matrix from a DICOM Network Server.

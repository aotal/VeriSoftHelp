Determining the Profile Position

  • In the matrix window, select the context menu Profile Position.
  • The Set Coordinate for Profile dialog will be displayed:
Set Coordinate for Profile dialog
  • Set the profile position by:
    • -selecting the point to use with a mouse click in the matrix.
    • -entering the coordinate in the dialog. Check the position by clicking the Show Position button.
  • Confirm the new profile position with the OK button.
  • The profile will be drawn through the nearest chamber center, the nearest voxel center, or the nearest pixel (refer to section Creating Profiles).

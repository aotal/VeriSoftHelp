Installation

Hint
  • On the VeriSoft installation CD/DVD, you will find the following installation programs:
  • -VeriSoft
  • -Track-it Import Client

To be able to export VeriSoft data to the Track-it database, it is imperative to additionally install the Track-it Import Client after you have installed VeriSoft.

Caution

Working with an incomplete version of VeriSoft.

Malfunction!

If a restart of the system is required during or after the installation of the software, it is essential to restart the PC before the first use of the software in order to ensure a correct installation of the software.

Hint

Switch off antivirus software, energy saving modes, and screen savers in order to avoid problems during installation or measurements.

The installation program is delivered on a CD/DVD.

  • Log in as the administrator.
  • Start Windows on your PC and close all running applications.
  • Insert the installation CD/DVD into the CD/DVD drive (e.g., drive D:\).
  • In Windows Explorer, open the directory of the CD/DVD drive.
  • Start the installation with the "setup.exe" file.
  • The installation program will run and the software will then install on the hard disk of your PC.
  • Please follow the prompts on the screen.
  • The Setup Type window will be displayed.
Setup Type window
  • In the Setup Type window, select the setup type Custom or Complete (or the VeriSoft software).

PTW-Freiburg recommends selecting the setup type Complete, especially if you use the software for the first time.
With Complete, all available programs will be installed in the directory 'C:\Program Files(x86)\'.

If you want to choose another directory, select the setup type Custom. The installation program recommends the target directory 'C:\Program Files(x86)\'.
To choose another directory, click Change.
With OK, you confirm the selection. The target directory will be created if it does not exist yet.

  • Click the Next button, and in the following window, click the Install button to start the installation.
  • VeriSoft will be installed.
  • After the installation is finished, the Program Registration window will be displayed.
Program Registration window
  • In the Program Registration window, enter the user information as well as the license key (see label of the installation CD/DVD). Confirm the entered data with the Register button.

If the data entered during installation are invalid, an appropriate message will be displayed. The Program Registration window will remain displayed to repeat the entering of registration data or to terminate registering the program by clicking the Demo or existing license button. In this case, a demo version of the software will be licensed, which can be used for a period of 30 days. The licensing information can be entered at a later date via the PTW License Control Center (see section Licensing Procedure).

If you had already installed a previous version with valid licensing, abort the registration process by clicking the Demo or existing license button. In this case, the licensing will remain also valid for the already installed version. Please keep in mind that this only applies if the major release numbers of the previous and of the new version are identical.

  • Install the Track-it Import Client if you want to export VeriSoft data to the Track-it database.

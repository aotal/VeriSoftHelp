Merging the Depth Dose Curves into One PDD File

  • Merging depth dose curves into one PDD file is only possible if these meet the following conditions:
  • -Depth dose curves must be available for at least 3 field sizes ≤ 10 cm.
  • -The measuring parameters modality, energy, wedge filter, block, SSD, gantry angle, collimator angle, as well as collimator offset in crossplane and inplane direction must be the same for all depth dose curves.

 

  • Select the menu Tools → Options and the 4D Dosimetry tab.
  • In the Generate PDD File pane, click the Select folder button.
  • In the standard Open dialog, select the directory with the depth dose curves.
  • All depth dose curves of the selected directory will be listed.
Generate PDD File pane with loaded depth dose curves
  • If required, add further depth dose curves from other directories by clicking the Add File(s) button.
  • Select depth dose curves that are not required by clicking them (blue background). Remove these depth dose curves from the display by clicking the Remove file(s) button.
  • Remove all depth dose curves from the display by clicking the Clear button.
  • Select the check boxes of the depth dose curves that you wish to merge into one PDD file.
  • Click the Merge and complete PDD file button.
  • If all conditions are met, the PDD file will be created and saved in the directory defined in the PDD Directory pane. Otherwise, an error message will be displayed.

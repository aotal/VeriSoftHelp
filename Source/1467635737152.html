Adjusting Compare Parameters

Gamma 2D and Gamma 3D Method

With the Definition of referenced max. dose for dose volumes list box, you can define the reference point for the comparison of dose volumes: the maximum value of the entire dose volume (default) or the maximum value of the currently displayed slice.

With the Gamma Index calculation list box, you can activate or deactivate the optional enhanced criteria (second and third pass) for the gamma index calculation, which was elaborated by Tom Depuydt et al. (Radiotherapy and Oncology 62 (2002) 309-319). PTW recommends using this option.

Furthermore, you can define the following compare parameters for the "Gamma Index" method:

  • -mm Distance-To-Agreement:
    accepted spatial deviation
  • Hint

    If a dose volume was loaded as the comparative matrix and the Gamma 3D comparison mode was selected, VeriSoft will check the accepted spatial deviation in all spatial directions.

  • -% Dose Difference with ref. to:
    accepted percentage dose deviation
  • -Reference value for the accepted percentage dose deviation:
    • -Local dose:
      dose at the corresponding position of the reference matrix
    • -Max. dose of ...:
      maximum dose of the reference matrix that depends on the setting in the Definition of referenced max. dose for dose volumes list box
      • -entire volume:
        maximum dose of the entire dose volume
      • -selected slice:
        maximum dose of the selected slice
    • -Selected dose:
      dose at an arbitrary point of the reference matrix
      Enter the desired dose value.
  • -Use increased tolerance:
    You can define a higher tolerance to be used below a defined threshold. For this, you have to select the appropriate check box and enter the higher tolerance and desired threshold value.
  • -Suppress doses below:
    You can determine not to perform the Gamma calculation if the dose values of the reference matrix are below a defined threshold. For this, select the appropriate check box and enter the threshold value as a percentage of the defined maximum dose. The maximum dose depends on the setting in the Definition of referenced max. dose for dose volumes list box.
    The points excluded from the evaluation will be labeled as not evaluated, displayed with a gamma value of 0 in the diagram, and marked with "---" in the table. The points excluded from the evaluation are not considered in the statistics and in the histogram.
    PTW recommends selecting this option to eliminate the effects of scattered radiation on the result.

'Local Percentage Difference' method and 'Difference in % of Normalization Value of Reference Matrix' method

For these two compare modes, you can define not to perform the calculation if the dose values of the reference matrix are below a defined threshold. For this, select the check box and enter the threshold value as a percentage of the maximum of the measured slice.
The points excluded from the evaluation will be labeled as not evaluated, displayed with a compare value of 0 in the diagram, and marked with "---" in the table. The points excluded from the evaluation are not considered in the statistics and in the histogram.

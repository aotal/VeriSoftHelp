Inclinometer Settings

Advanced tab of the Measurement Options dialog

The Advanced tab is available only when the selected accessory is the Inclinometer.

On this tab, you will find the following options:

Gantry Upright Position

  • Choose the upper upright position of the gantry:
  • -0°
  • -180°

Gantry Rotation Direction

  • Choose the rotation direction of the gantry:
  • -Clockwise
  • -Counter-clockwise

Use angle correction table

If this option is selected, you can apply an angle correction, using correction tables. For further information on creating correction tables for angle correction, please refer to Correction Tables for Angle Correction in Case of Inclinometer Measurements.

Display of Isodoses

If a dose distribution is available in the loaded data set, you can display it in addition to the patient anatomy.

  • For this, select the appropriate check boxes in the Select visible Isodoses pane (2c).
  • The selected isodoses will each be displayed as an isodose line in the corresponding color.
    In 2D view, you can display the isodoses as isodose areas if you enable the Calculated Dose representation in the list box (1b).
     

In the Select visible Isodoses pane (2c) you can, furthermore, determine the value to which the isodoses have to be normalized:

  • -Selected Isocenter
    Normalization to the dose value at the selected isocenter (default)
  • -Norm. Value of Data Set A/B
    Normalization to the normalization value of data set A or B
    This option is only available if a data set is loaded in the corresponding matrix window.
  • -Selected Value
    Normalization to a user-defined value
    Enter the desired value and confirm it with the Apply button.

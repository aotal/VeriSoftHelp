Zeroing

When starting a measurement for the first time, the system must first be zeroed. The zeroing will be automatically performed for all available measuring ranges. To do so, select the menu Measurement → Zero. A message window is displayed while zeroing is in progress.

Two times can be set for the zeroing procedure in the menu Tools → Measurement Options on the Measurement tab (also refer to section Measurement Settings).

  • -Zeroing Time duration of a zeroing procedure
  • -Zeroing Reminder validity period that determines the zeroing intervals

Validity

Each time a measurement is started, the program checks whether the last zeroing procedure is older than the set validity. If yes, VeriSoft requests a zeroing procedure.

Hint

We recommend zeroing the system each time the detector interface is turned on. VeriSoft does not check whether or not the detector interface was switched off. Allow the devices to warm up and stabilize for the time specified in the respective user manuals.

Tabular View of the Comparison

The tabular view of the comparison shows the analysis according to the dose areas.

A dose area contains all the voxels whose dose values are higher than or equal to the appropriate dose level. The dose level is specified in % of the normalization value of the reference data set.
Example:
The dose area for dose level 50 contains all the voxels whose dose is ≥ 50 % of the normalization value of the reference data set.

The view consists of the following six columns:

  • -Column 1 = Dose level for evaluation ...
    Selection of the dose range to be displayed by selecting or clearing the check box of the appropriate dose level.
    The following dose levels are available by default:
    10, 30, 50, 70, 80, 85, 90, 95, 100, 105, 110, and 120
    The dose levels 10, 50, and 100 are activated by default.
  • -Column 2 = Number of Voxels
    Number of voxels whose dose is higher than or equal to the corresponding dose level
  • -Column 3 = Evaluated Voxels
    Number of evaluated voxels whose dose is higher than or equal to the corresponding dose level
    Voxels will be evaluated if they are above the threshold defined in the Compare pop-up window of the compare window (refer to section Selecting the Compare Mode and Adjusting the Compare Parameters).
  • -Column 4 = Passed
    Number of Evaluated Voxels that have met the adjusted criteria.
    The percentage ratio to the Evaluated Voxels is indicated in brackets.
  • -Column 5 = Failed
    Number of Evaluated Voxels that have not met the adjusted criteria.
    The percentage ratio to the Evaluated Voxels is indicated in brackets.
  • -Column 6 = Result
    Result according to the evaluation criteria adjusted in the menu Tools → Options on the Compare tab (refer to section Compare Mode Options).

 

  • By selecting the context menu Copy, you can copy the table to the clipboard. From the clipboard, you can paste it to other software.
  • By selecting the context menu Save As, you can save the table as a CSV file (*.csv) or as a text file (*.txt).

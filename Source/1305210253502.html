Composing and Loading Matrices

Notice

To avoid equipment damage when measuring large fields, measure the matrices on the Gun side with the detector array reversed. Thus the electronics will not be irradiated.

The Compose function considers this turning and will display the matrices on the Gun side correctly.

Hint

This function is available for detector array matrices only.

Composing and loading matrices is only possible if they have been measured with the same device.

Hint

It is not possible to compose and load dose volumes (e.g., xcc files of OCTAVIUS 4D measurements). If this is attempted, an error message will be displayed.

The Compose function at loading matrices is used to generate detector array matrices for bigger field sizes.

For this, 2 or 4 matrices will be composed on the basis of the selected rows and columns. The resolution of the composed matrix will not be changed.

At the loading, these matrices will be composed. The composed matrix can be saved in VeriSoft as a detector array matrix.

Hint

Composing only combines the measuring points of all the loaded measurements without changing the measured values.

  • The matrices will be designated as follows:
  • -matrix #1:
    detector array measurement Target-Left
  • -matrix #2:
    detector array measurement Gun-Left
  • -matrix #3:
    detector array measurement Target-Right
  • -matrix #4:
    detector array measurement Gun-Right
  • Composing of matrices is only possible for the following combinations of matrices:
  • -matrix #1 and #2
  • -matrix #1 and #3 or
  • -matrix #1, #2, #3, and #4

 

  • To compose and load matrix A or matrix B, select menu File → Data Set A/B → Compose.
  • The Open Data to Compose dialog will be displayed (refer to Fig. Open Data to Compose dialog).
  • Open matrix #1 by clicking the Open #1 button.
  • The grayscale image will be displayed in the button. By moving the mouse cursor over the image, a tooltip with the path and name of the file will be displayed.
  • Open the matrix #2 or the matrix #3 or the matrices #2, #3, and #4 by clicking the appropriate buttons.
  • The grayscale images will be displayed in the appropriate buttons. By moving the mouse cursor over the images, tooltips with the path and name of the appropriate files will be displayed.
  • For the matrices on the left and on the right side, enter the columns, which shall be used for the composed matrix.
  • For the matrices on the Gun and on the Target side, enter the rows, which shall be used for the composed matrix.
  • Confirm the selected matrices and the entries with the OK button.
  • The matrices will be composed and loaded.

For further information on loading matrices, please refer to section General Information on Loading.

Zoom Function

If two matrices are loaded, you can evaluate a de-fined area of the loaded matrices by means of the Zoom function. By using the Zoom function, the original matrices will not be changed.

  • Select the Zoom entry in the list box in the toolbar.
  • In the results window, the grayscale image of matrix B will be displayed.
  • Hold the left mouse button down and draw a rectangle to define the area to zoom.
  • The selected area will be marked by a red rectangle.
  • The other windows will show the selected area and all evaluations will be recalculated for this area.
  • You can always change the area to zoom by drawing another rectangle.
Hint

In Zoom mode, you can toggle to the profile representations.

Performing image processing is not possible in Zoom mode.

  • Return to the original view of the matrices by:
    • -selecting again the Zoom entry in the list box in the toolbar and
    • -selecting the context menu Reset in the Zoom representation.

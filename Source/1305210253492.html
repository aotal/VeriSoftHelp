Loading Lists of Matrices

Hint

This function is not available for dose volumes.

Instead of individual matrices or dose volumes you can load lists of matrices.

  • To load the list of matrices A or B select menu File → Data Set A/B → Open List.
  • The Define List and Sequence dialog (refer to Fig. Define List and Sequence dialog ) will be displayed.
  • Click the Select Folder button.
  • Select in the standard Open dialog the directory with the matrices to load.
  • All files of the selected directory will be inserted into the list.
  • Click the OK button.
  • The list will be loaded and the first matrix of the list will be displayed in the appropriate matrix window.

For further information on loading matrices, please refer to section General Information on Loading.

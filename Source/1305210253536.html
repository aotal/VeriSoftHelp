Cross Calibration

Instead of performing an air density correction, you can correct the measured values by means of the cross calibration. In case of a cross calibration, the temperature, air pressure, energy, and output of the radiation device will be corrected at once.

Hint

PTW recommends repeating the cross calibration each working day.

PTW recommends performing the cross-calibration with a field size matching the field size of the application. A typical field size for the OCTAVIUS Detector 729 or the OCTAVIUS Detector 1500 is 10 cm x 10 cm. A typical field size for the OCTAVIUS Detector 1000 SRS is 4 cm x 4 cm.

  • Select the Cross Calibration option in the Measurement Parameters dialog:
Measurement Parameters dialog – Cross Calibration
  • In the Expected value input box, enter the measured value expected for the reference measurement of the center chamber (e.g., 1 Gy).
  • Start a measurement by clicking the Measure button.
  • The measuring window will be displayed.
  • Perform the desired reference measurement.
    Therefore, irradiate a reference field (e.g., 10 cm x 10 cm, 1 Gy). The irradiated dose has to comply with the entered dose value in the Expected value input box.
Hint

For further information on performing a measurement, refer to section VeriSoft Measuring Window.

  • Accept the measuring data by clicking the Accept button.
  • The actual measured value will be displayed in the Measured value box and the correction factor for cross calibration will be calculated.
Hint

The reference measurement is not a normal measurement. It is only used to determine the correction factor for cross calibration.
All further measurements are corrected by multiplication with the correction factors kCross and kUser.
In this case, an air density correction by entering temperature and air pressure is not possible.

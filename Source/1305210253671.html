Changing Colors and Isodose Levels

Via the menu Graphics → Edit Contour Settings, the Graphic Colors and Isodose Levels dialog will be displayed.

Graphic Colors and Isodose Levels dialog
  1. 1Tabs for the settings for matrix A and matrix B
  2. 2Check box to select or deselect the isodose level
  3. 3Button for the selection of the isodose color from the standard Windows color dialog
  4. 4Isodose level
  5. 5Button to restore the default settings of isodose colors and isodose levels
  6. 6Button to sort the isodose levels in descending order
  7. 7Button for the selection of the profile color from the standard Windows color dialog
  8. 8Button for the selection of the profile line style
  9. 9Input box for profile line thickness

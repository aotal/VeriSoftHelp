Compare Mode Options

Compare tab of the Options dialog

On the Compare tab, you will find the following options:

Comparison Settings

With the Gamma Index calculation list box, you can activate (default) or deactivate the optional enhanced criteria (second and third pass) for the gamma index calculation, which was elaborated by Tom Depuydt et al. (Radiotherapy and Oncology 62 (2002) 309-319). PTW recommends using this option.

With the Definition of referenced max. dose for dose volumes list box, you can define the reference point for the comparison of dose volumes: the maximum value of the entire dose volume (default) or the maximum value of the currently displayed slice. The Compare tab in the compare window changes accordingly (refer to section Gamma 2D and Gamma 3D Methods).

Result Settings

In the Result Settings pane, you can adjust the following settings separately for each compare mode:

  • -In the first line, you define the maximum compare value for the classification passed.
Hint

For the "Gamma Index" method, this value is fixed and it cannot be changed.

  • -In the second and third lines, you can enter threshold values for the evaluation of the calculated compare values.
    • If the criterion entered in the upper input box is met, the result under Results in the results window is marked with a green dot.
    • If the criterion entered in the lower input box is met, the result under Results in the results window is marked with a yellow dot.
    • If neither criterion is met, the result under Results in the results window is marked with a red dot.

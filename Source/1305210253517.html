Selecting a Matrix in a DICOM Directory

To select a matrix faster, you can use the input boxes for the last name or the ID of the patient as filters.

  • In the DICOM Import dialog, enter the last name or the ID of the patient. Enter the whole parameter or only the initial characters with an asterisk "*" at the end.
  • Click the Search button.
  • The tree view will show only the DICOM patient data sets that meet the selected search criteria.

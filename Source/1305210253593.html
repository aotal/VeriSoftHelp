Undo and Redo Processing

Via the menu Edit → Undo you can undo one or several process steps.

Hint

The functions "Defining the region of interest (ROI)" as well as "Rotating through an arbitrary angle (Rotate)" cannot be reversed.

In case of undoing the process step "Subtracting a background value (Background)", you may not achieve the original matrix as the negative values will be set to 0 when subtracting a background value.

If you have reversed the process steps by means of the Undo function, you can restore them via the menu Edit → Redo.

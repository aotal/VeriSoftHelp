Calibration of Non-Detector Array Matrices

  • When moving the mouse cursor over the Edit tab, a pop-up window with buttons will be displayed on the left side of the matrix. To calibrate a non-detector array matrix, click the appropriate button or select the menu Edit → Data Set A/B → Calibrate.
  • The Calibrate dialog will be displayed:
Calibrate dialog - non-detector array matrices
  • A non-detector array matrix can be calibrated in different ways:
  • -by multiplication with a factor
  • -by modifying the unit.

A non-detector array matrix can be calibrated more than once.

Multiplication with a Factor

All values of the image are multiplied by a factor.

  • Determine the appropriate multiplication factor:
  • For one loaded matrix:
    If you move the mouse inside the matrix, you can read the dose values in the status bar. By clicking a selected point, the dose value of this position is entered as the numerator of the factor.
  • For two loaded matrices:
    By clicking a selected point in the matrix, the dose value of this position will be entered as the denominator of the factor. The dose value of the corresponding position within the other matrix will be entered as the numerator of the factor. By multiplication with this factor, the same dose value will be assigned to the selected position in both matrices.
  • Confirm the entry with the OK button.
Hint

After the calibration, the normalization values are recalculated.

Hint

If a dose volume is loaded, all the dose values of the volume will be multiplied with the factor.

Changing the Unit

The unit of a non-detector array matrix is loaded with the matrix (if a unit exists) and displayed in the Current Unit field.

  • Select the new unit from the New Unit list box and confirm the selection with the OK button.
Hint

The "Change Unit" function only replaces the unit. The values will not be recalculated.

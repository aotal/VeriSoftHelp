Storage of Data

When you store data, be sure to write them only to shared directories. Otherwise, the data will be stored in a virtual directory ("virtualization"). These data in the selected storage path will no longer be visible to other users or in Windows Explorer.

If corresponding PTW-specific files exist, they will be saved to the following directories:

  • <PTW application files>
    e.g., configuration files or initialization files

    • Directory:
    'C:\ProgramData\PTW'

  • <PTW documents>
    e.g., demo data, measurement data, or evaluation data

    • Directory:
    'C:\Users\Public\Public Documents\PTW'

Selection of the Measuring Devices

Devices tab of the Measurement Options dialog

On the Devices tab, you will find the following options:

Detector Array

  • List box for selecting the detector array for the measurement
    The following detector arrays can be selected:
  • -OCTAVIUS Detector 729
  • -OCTAVIUS Detector 729 XDR
  • -OCTAVIUS Detector 1000 SRS
  • -OCTAVIUS Detector 1500
  • -OCTAVIUS Detector 1500 MR
  • -OCTAVIUS Detector 1500 XDR
  • -2D-ARRAYseven29
  • -2D-ARRAY XDR
  • -Device Demo

If the Device Demo detector array is selected, there will be no communication with the hardware. The measurement will be simulated by VeriSoft.

If more than one detector array of the same type is used, PTW recommends enabling the Use multiple detectors of same type option. In this case, when you close the Measurement Options dialog, you will be prompted to verify that the selected calibration file and the detector array used correspond.

Calibration File

Display of the currently set calibration file for the selected detector array

 … 
Button for selecting the calibration file.

At the installation of VeriSoft, the calibration file (*.cal) supplied on the installation CD is copied to the following folder (refer also to section Storage of Data):
'<PTW documents>\Common Files\Calibration Files'

If you select a calibration file in which measuring chambers are deactivated, an appropriate message will be displayed. For more information about the deactivation of measuring chambers, please refer to the user manual of the PTW software ArrayCal.

Accessories

  • List box for selecting accessories for the measurement
    The following accessories are selectable:
  • -Rotation Unit
    Standard version or modular version of the rotation unit in combination with the OCTAVIUS standard top for performing OCTAVIUS 4D measurements
  • -Rotation Unit SRS
    Modular version of the rotation unit in combination with the OCTAVIUS SRS top for performing OCTAVIUS 4D measurements with a small diameter
  • -Rotation Unit planar
    Modular version of the rotation unit in combination with the OCTAVIUS Linac QA top for measuring detector array matrices
  • -Inclinometer
  • -None (no accessory)
  • -Rotation Unit Demo
  • -Inclinometer Demo

If the selected accessory is a Demo type accessory, there will be no communication with the hardware. The measurement will be simulated by VeriSoft.

Connections

Display of the current settings for data transfer

The data transfer parameters need to be set separately for the selected detector array and for the selected accessory.

You start the search by clicking the Search button. Then, the Devices dialog with all connected devices will be displayed. Once you select a device, the corresponding data transfer parameters will automatically populate the corresponding field in the Measurement Options dialog.

Alternatively, you can adapt the settings for data transfer manually by means of the Edit buttons.

For further information on data transfer settings, please refer to section Settings for Data Transfer.

Reduce RS232 Baudrate

If the selected detector array is connected via an RS232 connection, the Reduce RS232 Baudrate option will be displayed.

If communication problems occur, e.g., when using long cables, you should select this option. For further information, please refer to section Error Messages / Troubleshooting.

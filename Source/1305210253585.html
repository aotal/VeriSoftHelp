Calibration of Detector Array Matrices

  • When moving the mouse cursor over the Edit tab, a pop-up window with buttons will be displayed on the left side of the matrix. To calibrate a detector array matrix, click the appropriate button or select the menu Edit → Data Set A/B → Calibrate.
  • Depending on the settings at the time of the measurement, the appropriate Calibrate dialog will be displayed:
Calibrate dialog – detector array matrix air density correction
Calibrate dialog – detector array matrix cross calibration
  • A detector array matrix can be calibrated in different ways:
  • -by modifying the air density correction parameters (refer to section Air Density Correction)
  • -by modifying the additional, free correction factor kUser (refer to section Setting Measurement Parameters)
  • -by modifying the unit.

A detector array matrix can be calibrated more than once.

Hint

The correction factor kEnergy for the beam quality cannot be changed via the Calibrate function.

Hint

If a dose volume is loaded, the calibration is applied to all the dose values of the volume.

Changing the Unit

The unit of a detector array matrix is loaded with the matrix (if a unit exists) and displayed in the Current Unit field.

  • Select the new unit from the New Unit list box and confirm the selection with the OK button.
Hint

The "Change Unit" function only replaces the unit. The values will not be recalculated.

Grayscale Images and Isodose Representations

If the grayscale or isodose representation of the matrices is selected, the matrix window offers the following options:

  • If one of the loaded matrices is a detector array matrix, you can project the positions of the chambers on the loaded matrices by selecting the menu View → Overlay Measuring Positions.
  • You can display or hide the legends to the matrices by selecting the menu Graphics → Legend.
  • You can set the isodose colors and isodose levels by selecting the menu Graphics → Edit Contour Settings (refer to section Changing Colors and Isodose Levels).
  • You can change the 3D isodose representation by means of the mouse, e.g., rotate in any direction by clicking and holding the left mouse button and moving the mouse or enlarge by clicking the right mouse button and moving the mouse.
  • By selecting the context menu Copy, you can copy the actual matrix representation to the clipboard. From the clipboard, you can paste it to other software.
  • By selecting the context menu Save As, you can save the actual matrix representation as a graphic file. The possible file formats are *.png, *.jpg, *.tif and *.bmp.

Compare Result

If two matrices are loaded, you can display a table with the compare result. For this, select the Results entry in the list box in the toolbar.

Both matrices must have the same unit. Otherwise, an appropriate message will be displayed and you have to calibrate the matrices first (refer to section Calibration).

  • The displayed compare result depends on the following settings:
  • -compare mode selected in the compare window
  • -evaluation criteria adjusted in the menu Tools → Options on the Compare tab (refer to section Compare Mode Options)

After you have selected a compare class in the compare window on the Compare tab, the Export to Track-it button will be activated and you can export the compare result to the Track-it database. For further information, refer to section Export to the Track-it Database.

  • By selecting the context menu Copy, you can copy the compare result to the clipboard. From the clipboard you can paste it to other software.
  • By selecting the context menu Save As, you can save the compare result as a CSV file (*.csv) or as a text file (*.txt).

The Results representation contains the following data:

Statistics Area

  • -Number of Dose Points
    Total number of matrix points.
  • -Evaluated Dose Points
    Number of evaluated points, i.e., total number minus the number of points below a defined threshold. You can set this threshold in the Compare pop-up window of the compare window (refer to section Selecting the Compare Mode and Adjusting the Compare Parameters).
  • -Passed
    Number of Evaluated dose points that have met the adjusted criteria.
    The percentage ratio to the Evaluated dose points is indicated in brackets.
  • -Failed
    Number of Evaluated dose points that have not met the adjusted criteria.
    The percentage ratio to the Evaluated dose points is indicated in brackets.
  • -Result
    Result according to the evaluation criteria adjusted in the menu Tools → Options on the Compare tab (refer to section Compare Mode Options).

Gamma 2D, Gamma 3D, Local %Difference and DoN Area

  • -Arithmetic Mean:
    arithmetic Mean of the calculated compare values
  • -Min:
    minimum calculated compare value with coordinates and value of the reference matrix
  • -Max:
    maximum calculated compare value with coordinates and value of the reference matrix
  • -Median:
    median of the calculated compare values

Absolute Difference Area

  • -Arithmetic Mean:
    arithmetic Mean of the dose differences
  • -Min:
    minimum calculated dose difference with coordinates and value of the reference matrix
  • -Max:
    maximum calculated dose difference with coordinates and value of the reference matrix
  • -Median:
    median of the dose differences

Settings Area

Settings in the menu Tools → Options on the Compare tab (refer to section Compare Mode Options)

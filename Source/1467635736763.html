Opening a Project

  • To open a project, select menu File → Project → Open.
  • If the password protection is activated in VeriSoft, enter the appropriate password. After starting the program, the password needs to be entered only once.
  • Select the desired project from the Open dialog.
Caution

Changing the VeriSoft settings by opening a project.

Incorrect Operation!

When a project is opened, the project settings will replace the current VeriSoft settings, which may lead to incorrect compare results.

  • In the prompt, click the Yes button if you want to open the project and overwrite the VeriSoft settings with the project settings.
    Click the No button to cancel opening of a project.
    By clearing the appropriate check box in the prompt, you disable the prompt until VeriSoft is restarted.
  • The matrix windows A and/or B will display the linked data. All changes performed in the Edit menu will be repeated automatically.
  • If the project contains an OCTAVIUS 4D measurement, the OCTAVIUS 4D dose volume will be calculated and displayed according to the stored settings.
  • In the result and compare windows, the comparison will be calculated according to the stored settings and the result will be displayed.

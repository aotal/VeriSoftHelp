Loading and Displaying Dose Volumes and Lists of Matrices

  • To load a dose volume, click the appropriate button or select the menu File → Data Set A/B → Open.
  • After loading a dose volume, the slice at a depth of 0 mm will be displayed in the matrix window.

  • To load a list of matrices, select the menu File → Data Set A/B → Open List.
  • After loading, the first matrix of the list will be displayed in the matrix window.

For further information on loading matrices, please refer to section Loading Matrices.

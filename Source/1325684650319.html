Importing Matrices from a DICOM Directory

  • Select the DICOM Dir import mode in the DICOM Import dialog.
  • Select the directory containing the DICOM data to import.
  • In order to be able to import the DICOM data, the selected directory must contain a DICOMDIR index file. Otherwise, the following inquiry will be displayed:
Inquiry in case of missing DICOMDIR file
  • By clicking the OK button, a temporary DICOMDIR will be created.
    By clicking the Cancel button, the inquiry will be closed and importing the DICOM data is not possible.
  • Select and import the desired data as described in sections Selecting a Matrix in a DICOM Directory and Importing a Matrix.

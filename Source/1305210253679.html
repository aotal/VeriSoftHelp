Advanced Options

Advanced tab of the Options dialog

On the Advanced tab, you will find the following options:

Miscellaneous

In this pane, you can define the Average Region for the determination of the measuring value at the cursor position, the normalization value, or the background value.

Choose one of these options:

1 pixel:
The measuring value is the value of the current pixel.
5 pixels:
The measuring value is the average value of the current pixel and the pixels to the right, to the left, above, and below.
9 pixels:
The measuring value is the average value of the current pixel and the eight neighboring pixels.

If the Show Patient ID check box is selected, the patient ID will be added to the matrix type. The patient ID is taken from the loaded file.

Appearance

In this pane, you can change the color scheme and the background color of VeriSoft.
The default settings are Scheme Black and gray background.
You can restore the default settings with the Default Scheme and Default Background buttons.

Security

The Security pane accommodates the Set Administration Password button for displaying the password administration.

In the password administration, you can activate or deactivate the password protection. Furthermore, you can set or change the required password.

For detailed information on the password administration, please refer to section Password Management.

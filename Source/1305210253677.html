Options for Loading Matrices

Data tab of the Options dialog

On the Data tab, you will find the following options:

Data Import

If you have selected the Convert Dose to check box and the desired unit in the list box, VeriSoft will check at the loading whether the matrix can be converted into the desired unit.
If a conversion is possible, VeriSoft will convert the unit automatically.
If a conversion is not possible, the matrix will be loaded without conversion.

If the conversion to cGy or mGy has been selected, the dose values for the Gamma 2D or 3D method entered in the compare window on the Compare tab (refer to section Gamma 2D and Gamma 3D Methods) will also be converted to cGy or mGy.

If one of the Activate data processing batch check boxes is selected, VeriSoft will display the dialog for selecting and executing the batch file when loading the appropriate matrix (refer also to section Working with the Batch Mode).

OCTAVIUS Detector 1500 2D-Data

For matrices measured with an OCTAVIUS Detector 1500, the profile resolution is 10 mm.

When the Profiles and tables only contain measured values check box is cleared (default setting), additional values between the measured values will be determined by interpolation in order to obtain a display resolution of 5 mm. The interpolated values will only be shown in the dose value table in the matrix window, in the profiles in the results window, and in the dose value table in the results window.
The interpolated values will not be considered for analysis.

When the Profiles and tables only contain measured values check box is selected, only the measured values will be shown.

Hint

A change of this selection becomes effective only after the matrices are reloaded.

General

If the Warn about unsaved data check box is selected and you invoke a command that would delete the currently displayed, unsaved data (e.g., loading matrices, exiting VeriSoft), you will receive a warning and you can save the data.

Finding a Matrix on a DICOM Network Server

  • Define the desired search criteria in the DICOM Import dialog.
    In the input boxes for the first and last names or the ID of the patient, you can enter the full parameter, the initial characters followed by an asteriks "*", or only an asterisk "*" in order to find all patients of the selected period.
  • Click the Search button.
  • The tree view will show all DICOM patient data sets that meet the selected search criteria.

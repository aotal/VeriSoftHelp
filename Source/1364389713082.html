Calculating PTW DVH Curves

Hint

If the following types of OCTAVIUS 4D measurement data are loaded as the reference matrix, it is not possible to calculate PTW DVH curves. An appropriate error message will be displayed.

  • -measurements for non-coplanar irradiation
  • -measurements for irradiation of a rotation unit not positioned at the isocenter
  • -composed measurements
  • In the matrix window A or B, load the OCTAVIUS 4D measurement according to section Loading OCTAVIUS 4D Measurements and Calculating the OCTAVIUS 4D Dose Volume.
  • To import the corresponding TPS data from a DICOM directory or from a DICOM network server, click the appropriate button.
  • The data to be imported must include the following files. Otherwise, it is not possible to calculate DVH curves.
    • -RTStruct, RTDose, and RTPlan file of the patient plan
    • -patient CT
  • To import the desired data set, proceed as described in section Importing Matrices in the DICOM Format.
    Furthermore, observe the settings required for the import from a DICOM network server according to section Settings for Importing DICOM Data.
  • The TPS data will be loaded and the available structures will be displayed in the ROIs pane.
    VeriSoft will check for which structures PTW DVH curves can be calculated. A calculation is only possible for structures that are inside the phantom volume, as measured values are only available for these structures. All other structures that are outside the phantom volume are dimmed.
Hint

When no OCTAVIUS 4D measurement is loaded, all structures are dimmed.

  • You choose the PTW DVH curves to be calculated by selecting the appropriate structure check boxes.
    You choose all PTW DVH curves that can be calculated by selecting Select All from the context menu.
    You deselect all PTW DVH curves that can be calculated by selecting Unselect All from the context menu.
  • The TPS DVH curves will be calculated on the basis of the existing patient CT and dose information in the RTDose file and will be represented by solid lines.
  • You calculate the PTW DVH curves from the OCTAVIUS 4D measurement data by clicking the appropriate button.
  • The PTW DVH curves will be calculated.
    Couch models are not taken into account in these calculations.
    PTW DVH curves will only be calculated on the basis of the values measured in the phantom, taking the patient CT into account. A density override of the patient CT registered in the RTStruct object during treatment planning will not be considered. For further information on the algorithm, please refer to the PTW White Paper D913.200.00 (Dose reconstruction in the OCTAVIUS 4D phantom and in the patient without using dose information from the TPS).
  • The calculated PTW DVH curves will be represented by dashed lines.
  • The calculated PTW DVH curves will automatically be saved to the directory set in menu Tools → Options on the DVH tab under Data Path.
  • If needed, enlarge the DVH window to the size of the program window by clicking the Maximize button.

Import a Patient Anatomy from the DICOM Network

  • To import a patient anatomy from the DICOM network server, click the appropriate button.
    Observe the settings required for the import according to section Settings for Importing DICOM Data.
  • The Load Remote Dataset dialog will be displayed (refer to Fig. Load Remote Dataset dialog).
  • Search the desired data according to section Searching a Patient Anatomy on the DICOM Network Server.
  • Import the desired data according to section Selecting and Importing a Patient Anatomy from the DICOM Network Server.
  • The patient anatomy will be loaded and displayed in the RTView window.
  • The isocenter will be marked with a red point.
  • In order to make all representation options available, enlarge the window to the size of the program window by clicking the Maximize button.

Subtracting Background Value

  • When moving the mouse cursor over the Edit tab, a pop-up window with buttons will be displayed on the left side of the matrix. To subtract a background value, click the appropriate button or select the menu Edit → Data Set A/B → Background.
  • The Subtract Background dialog will be displayed:
Subtract Background dialog
  • Click anywhere on the grayscale image. A cross marker will be displayed and the dialog will show the current position and the corresponding measured value. You shift the cross marker by clicking another point of the image.
  • You can modify the coordinates of the cross marker via the dialog. Click the Apply button to move the cross marker to the new location.
  • Click the Goto Min button to move the cross marker to the position of the minimum value.
  • Irrespective of a selected point, the value to subtract can be entered in the dialog.
  • Confirm the selection with the OK button.
  • The background value will be subtracted from the values of all points of the matrix. Negative dose values will be set to 0.
Hint

Detector array matrix:
If the position of the background value of a detector array matrix is defined by a mouse click, the position and the value of the nearest chamber center will be displayed.

OCTAVIUS 4D matrix:
If the position of the background value of an OCTAVIUS 4D matrix is defined by a mouse click, the position and the value of the nearest voxel center will be displayed.

Non-detector array matrix:
In case of a non-detector array matrix the value for background distraction is determined by the so-called 'Average Region' that you can adjust in the menu Tools → Options on the Advanced tab (refer to section Advanced Options).

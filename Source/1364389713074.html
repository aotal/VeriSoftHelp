Settings for the Dose Volume Histograms (DVH)

DVH tab of the Options dialog

The DVH tab is only displayed when a licensed "DVH 4D" option is available.

Hint

Changes on the DVH tab will become effective only when DVH data are loaded again.

On the DVH tab, you will find the following options:

4D Dosimetry Patient

Input box for the voxel side length in [mm]
This value applies to the voxel side lengths in X, Y, and Z direction.
For the definition of the coordinate system, refer to section Display Planes of Dose Volumes.
valid range:
1.0 … 10.0
default:
2.5
Hint

PTW recommends setting the voxel side length in VeriSoft according to the values in the radiation treatment planning system.

Data Path

Selection of the directory to which the calculated PTW-DVH curves will automatically be saved and from which the calculated PTW-DVH curves will automatically be loaded
default:
'<PTW documents>\VeriSoft\Data\DVH'

Button for selecting a directory

Body Contour

The name of the body contour must be entered in the Body Contour Name input box. This name and the name in the TPS data (RTStruct object) must be identical. If this is not the case, an error message will be shown in the status bar of the DVH window and it will not be possible to calculate DVH curves.

Correction Value

In the kDVH input box, you can enter a correction factor for calculation of the DVH curves. The patient dose will be multiplied with this correction factor for calculation of the DVHs.
default:
1.0

Several factors may lead to a systematic minimal difference between the patient dose reconstructed by VeriSoft and the value calculated by the TPS. The two majors factors are the calibration method of the detector array and the relative electron density of the OCTAVIUS rotation unit assumed by the TPS. The resulting deviations are in the range of up to 3 %. If the detector array is always calibrated in the same way and the phantom density setting in the TPS is not changed, the systematic deviation is always the same. It can be compensated with the correction factor kDVH.

You can determine the correction factor kDVH as follows:

Compare the maximum dose values of the TPS DVH curves with the values of the VeriSoft curves for 10 plans that should be as diverse as possible. From the deviations calculate the average. If the average deviation is factor x, then enter the value 1/x as the correction factor kDVH.
The plans used for this calculation should include as little lung tissue as possible and only minimal volumes of air.

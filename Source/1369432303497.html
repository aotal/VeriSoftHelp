Displaying Calculated PTW DVH Curves

  • To import the corresponding TPS data from a DICOM directory or from a DICOM network server, click the appropriate button.
  • To import the desired data set, proceed as described in section Importing Matrices in the DICOM Format.
    Furthermore, observe the settings required for the import from a DICOM network server according to section Settings for Importing DICOM Data.
  • The TPS data will be loaded and the available structures are displayed in the ROIs pane.
    If PTW DVH curves were already calculated and saved for this data set, these will automatically be loaded as well.
  • To display the desired DVH curves, click the appropriate structure check boxes.
    To display all available DVH curves, choose Select All from the context menu.
    To hide the displayed DVH curves, choose Unselect All from the context menu.
  • The TPS DVH curves will be calculated on the basis of the existing patient CT and dose information in the RTDose file and will be represented by solid lines.
  • The calculated PTW DVH curves are represented by dashed lines.
  • If needed, enlarge the DVH window to the size of the program window by clicking the Maximize button.

Following-up the Image Processing

If you perform image processing for one or both of the matrices, the processed commands will be saved in the command stack. Commands that have been reversed via the Undo function will appear disabled.

You can display the command stack by selecting the menu View → Show Command Stack.

Show Command Stack dialog

In addition to the processed commands the assignment to the processed matrix will be displayed.

When closing or loading a matrix, the commands of the command stack concerning that matrix will be deleted. When exiting VeriSoft, the complete command stack will be deleted.

You can print out the command stack if you have selected the Data Origin check box in the Print dialog (refer to section Printing the Matrices and the Results).

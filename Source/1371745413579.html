Searching a Patient Anatomy on the DICOM Network Server

  • Define the desired search criteria in the Load Remote Dataset dialog.
    In the input boxes for the first and last name or the ID of the patient, you can enter the whole parameter, the initial characters with an asterisk "*" at the end, or only an asterisk "*" in order to search all patients in the selected period.
  • Click the Search button.
  • In the Patients pane, all DICOM patient data sets complying with the selected search criteria will be displayed in tree view.

DICOM import dialog

DICOM Import dialog - DICOM DIR
  1. 1Selection of the import mode
  2. 2Selection of the directory holding the DICOM data to import
  3. 3Input boxes for finding the patient via the patient's ID or the last name
  4. 4Selection of the modalities to display
  5. 5Check box for displaying the preview of the matrix
  6. 6Tree view showing the patients
  7. 7Options for the DICOM data import
  8. 8Display of the data depending on the highlighted level in the tree view (6)
  9. 9Preview of the matrix to import
    The preview will only be displayed if the Thumbnail check box is selected and the selected object contains a matrix.
DICOM Import dialog - DICOM SCP
  1. 1Selection of the import mode
  2. 2Name of the selected DICOM server (SCP server) in the network
  3. 3Input boxes for finding the patient via the patient's ID, first name, last name, or sex
  4. 4Input boxes for finding the patient by specifying the period when the data were created
    These periods are available for selection:
    • -All
      no limitation of the date range
    • -Today
    • -Today and yesterday
    • -Yesterday
    • -Last 7 days
    • -Last 14 days
    • -Custom date range
      You can define the period (Date Range), using the Start date and End date list boxes.
  5. 5Selection of the modalities to display
  6. 6Tree view showing the patients
  7. 7Options for the DICOM data import
  8. 8Display of the data depending on the highlighted level in the tree view (6)

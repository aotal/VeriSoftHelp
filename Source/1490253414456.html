Defining Dose Levels

  • In the tabular view pane, select the context menu Edit Dose Level.
  • The Dose Levels dialog will be displayed:
Dose Levels dialog
  • Define the appropriate dose levels.
  • Select the dose levels to be displayed by selecting the appropriate check boxes.
  • By clicking the Sort by value button, you can sort the dose levels.
  • By clicking the Set to default button, you can reset the dose levels to the default values.
  • Confirm the settings with OK.
  • The analyses for the set dose levels will be calculated and displayed in the tabular view of the Volume Analysis dialog.

Setting Measurement Parameters

  • To measure matrices, click the appropriate button or select the menu File → Data Set A/B → Measure.
  • The dialog for entering the measuring parameters will be displayed first:
Measurement Parameters dialog – Correction T and P
  • In this dialog, define the following measuring parameters:
    • -Correction parameters for air density correction or cross calibration (refer also to section Air Density Correction and Cross Calibration)
    • -Additional, free correction factor kUser
      valid range: 0.1 … 10
      default:1.0
    • -Correction factor kEnergy for beam quality for air density correction (refer also to section Correction Factor for the Beam Quality)
    • -Definition of isocenter position (also refer to section Defining the Isocenter Position).
      The definition of isocenter position is only required if the detector array is not positioned at the isocenter.
      If the selected accessory is Rotation Unit or Rotation Unit SRS, it will not be possible to define the isocenter position.
    • -Automatic saving of original values of the measurements:
      Select the appropriate check box. Click the  …  button and select a directory. Now the measurements will be stored automatically in this directory. The file names will consist of the current date and time.

By default the Measurement Parameters dialog is displayed before each measurement. If you clear the Show dialog on next measurement check box, the dialog will be displayed only before the first measurement after a restart of VeriSoft. Thus, you can define the parameters for all of the following measurements at once.

Hint

You can also display the Measurement Parameters dialog by selecting menu Edit → Measurement Parameters. The changed parameters will then affect the following measurements.

To change the correction values of displayed detector array matrices, you have to use the Calibrate function (refer to section Calibration of Detector Array Matrices).

  • Confirm the entered parameters with OK.

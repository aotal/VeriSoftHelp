Exporting Detector Array Matrices in the Special TIFF Format for TomoTherapy

This function is used for administrating the verification data in the TomoTherapy software.

  • To save detector array matrices in the special TIFF format for TomoTherapy, select menu File → Data Set A/B → Export → Tif - Hi•Art optimized.
  • The detector array matrix will be saved as a 16 bit TIFF grayscale image.
Hint

The saved TIFF image does not include all the measurement parameters anymore.

A white border will be added to the TIFF image and it will include two black cross markers; one in the left lower and one in the right upper corner. These markers comply with the detector array mounting holes outside the measurement area and can be used to match the TIFF image with a CT scan of the detector array.

Together with the TIFF image, a calibration file will be stored in the same directory with the same name but with the file extension *.cal. This calibration file can be used to calibrate the grayscale image, for example, on a TomoTherapy workstation.

Example of a calibration file *.cal
  • The calibration file includes 3 columns:
  • -The first column is for reference purposes only.
  • -The second column indicates the dose values in [cGy]. The dose value in the last row corresponds to the maximum dose of the detector array measurement.
  • -The third column indicates the 16 bit grayscale values corresponding to the dose values entered in the second column.

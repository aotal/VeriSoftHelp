Export to the Track-it Database

Hint
  • Before data can be exported to the Track-it database, the following software must be installed:
  • -Track-it
    For further information, refer to the Track-it user manual.
  • -Track-it Import Client
    For further information, refer to section Installation and to the Track-it Import Client user manual.
  • Using Track-it Import Configuration, configure the settings for the export to the Track-it database. For further information, refer to the Track-it Import Client user manual.
  • To export the compare result from the result window to the Track-it database, click the Export to Track-it button in the Results display (refer to section Compare Result).
  • To export the compare result of the dose distribution in the volume to the Track-it database, click the Export to Track-it button in the Volume Analysis dialog (refer to section Comparison of Dose Distribution in Volume).
Hint

You may be prompted to first log in with your user name and Track-it password. For further information on the login procedure, please refer to the Track-it user manual.

  • The data will be exported to the Track-it database and an appropriate message will be displayed.
  • In addition to the data, the following graphics will also be exported to the Track-it database:
    • -currently displayed matrix in matrix window A (e.g., currently set slice of a TPS dose volume)
    • -currently displayed matrix in matrix window B (e.g., currently set slice of a OCTAVIUS 4D dose volume)
    • -current graphic representation of the compare result in the compare window
    • -histogram comparing the dose distribution in the volume (only if exported via the Volume Analysis dialog)
  • When exporting from the Volume Analysis dialog, the tabular comparison of the dose distribution in the volume will be saved as a CSV file.
    A CSV file can be directly opened with a spreadsheet program.
  • Furthermore, a VeriSoft project will be created automatically in the Track-it database for each export. Also observe the information given in section VeriSoft Projects.

More Options

Advanced tab of the Options dialog

On the Advanced tab, you will find the following options:

Warn about unsaved data

When this option is selected and you invoke a command that would delete the current, unsaved measurement (e.g., starting a measurement), you will receive a warning and then you can save the measurement.

Warn about deactivated chambers

If you select a calibration file in which individual measuring chambers are deactivated, an appropriate message will be displayed. The display of the message can be deactivated from the message itself or by clearing this check box (Exception: deactivated center chamber). The display of the message can be activated by selection of this check box.

For more information about the deactivation of measuring chambers, please refer to the user manual of the PTW software ArrayCal.

Chamber Selection

By entering the row and the column, you define the measuring chamber of the detector array whose measuring value shall be displayed in the status bar.

The displayed diagram should help you define the correct row and column. The diagram shows the position of the first row and the first column.

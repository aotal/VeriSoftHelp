Selecting the Compare Mode and Adjusting the Compare Parameters

When moving the mouse cursor over the Compare tab, a pop-up window will be displayed on the left side of the compare window (refer to Fig. Compare pop-up window with parameters for the "Gamma Index" method ). On the Compare tab, you can now choose between two options for adjusting the compare parameters:

  1. Select a compare class in the list box at the top. For further information on determining compare classes, please refer to section Defining Compare Classes.
  • The compare result will be calculated and displayed according to the compare parameters set in the compare class.

  1. Select a compare mode in the second list box:
    • -Gamma 2D and Gamma 3D
      "Gamma Index" method
    • -Local%Difference
      "Local Percentage Difference" method
    • -DoN (%Diff ref Norm. Value)
      "Difference in % of Normalization Value of Reference Matrix" method
  2. Hint

    The Gamma 3D compare mode can be selected only when the loaded compare matrix is a dose volume.

  • Adjust the compare parameters for the selected compare mode according to section Gamma 2D and Gamma 3D Methods and section 'Local Percentage Difference' Method and 'Difference in % of Normalization Value of Reference Matrix' Method.
  • Click the Apply button (or press the Ctrl and A keys).
  • The compare result will be calculated and displayed.

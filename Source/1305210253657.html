Scrolling in Dose Volumes or Lists of Matrices

When moving the mouse cursor over the Slice tab, a pop-up window with a slider will be displayed on the left side of the matrix. With this slider, you can scroll through the slices of a dose volume or through the list of matrices. In matrix A you can also scroll with the PageUp and PageDown keys and in matrix B with the Shift key and the PageUp and PageDown keys.

  • The order of matrices shown corresponds:
  • -to the saved contents in case of dose volumes and
  • -to the order of the files in the list in case of lists of matrices.
  • If two dose volumes or lists of matrices are loaded (for matrix A and matrix B), you can set synchronous scrolling with Tools → Synchronous Data Set Scrolling.
  • -In case of dose volumes: by moving a slider, the slices with the same or the nearest depth are selected and shown in both matrix windows (as far as it is possible because of the loaded data).
  • -In case of lists of matrices: by moving a slider, the matrices with the same indices (not with the same offset) are selected and shown in both matrix windows (as far as it is possible because of the number of loaded files).
  • -In case of unsynchronized scrolling: only the active window is scrolled. The display of the other window remains unchanged.

Settings for Data Transfer

  • Data transfer is possible by the following communication channels:
  • -LAN Network Connection
  • -LAN Direct Connection
  • -RS232 Direct Connection
Hint

PTW recommends communicating via the LAN Network Connection.

To select the communication path for data transmission and to set the parameters required in each case, select menu Tools → Measurement Options and the Devices tab.

Printing the Matrices and the Results

  • To print the matrices and the results, select the menu File → Print.
  • To display the print preview of the matrices and the results, select the menu File → Print Preview.
  • First, the Print dialog will be displayed.
Print dialog (e.g., for the print preview)
  • Enter the administrative data for the printout or the print preview.
Hint

The patient ID and the patient name will be displayed automatically if they are available in the file.

  • Select the parameters to print or to display by selecting the appropriate check boxes.
    • -Data Origin:
      file names and paths of the loaded matrices as well as performed modifications
    • -Images:
      current contents of the graphic windows
    • -Results:
      compare results and settings
    • -Additional Statistics:
      additional statistic results
    • -Custom logo:
      If required, select a user logo by clicking the Select button.
    • -Confirmation:
      A line for the signature is added at the end of the page.
  • By means of the list box, select the output format for the printout or the print preview.
Hint

If an output format is selected for which no application is installed on the PC (e.g., output format PDF without the application Adobe Reader), you cannot display the print preview.

  • To define the page layout, click the Page Settings button.
  • To additionally save the print preview or the printout, select the Copy output to folder check box. Click the  …  button and select the desired folder.
  • To print the data, click the Print button or click the Preview button to display the print preview.
Hint

The printout is automatically written in French, if you have set the French language in the control panel.

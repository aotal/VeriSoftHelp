Changing the Grayscale Display

This representation option is only available for 2D and DRR representation.

The CT values will be displayed as grayscale values. The grayscale display can be changed by means of the two sliders (2f) below the patient anatomy.

  • -Width:
    With the Width slider, you define the range of CT values to be displayed, i.e., the bandwidth. All values below the range will be displayed black and all values above will be displayed white.
  • -Level:
    With the Level slider, you determine the middle value of the range.
     

You can reset the grayscale display to default:

  • 2D view:
    Click the Reset 2DView button (2e).
  • DRR view:
    Select again the DRR view in the list box (1a).

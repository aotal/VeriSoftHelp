Loading Measurements with Detector Array and Inclinometer

(without rotation unit)

In case of measurements with the detector array and inclinometer, the gantry angle and the dose values of the measuring chambers of the detector array will be saved for each measurement interval. For further information please refer to section Measurement with Detector Array and Inclinometer.

When loading the data, the dose values of the measuring chambers will be added.

 

  • To load matrix A or matrix B, click the appropriate button or select menu File → Data Set A/B → Open.
  • Select the desired xcc file in the standard Open dialog.
Hint

The appropriate mcc file contains only the dose values. The gantry angles are only available in the xcc file (refer also to section Saving Measurements Performed with Detector Array and Planar Rotation Unit or Inclinometer).

  • The Select an angle range dialog will be displayed:
Select an angle range dialog for gantry angles

In the Select Range pane you determine whether the entire measurement or the measured values of a defined angle or time range shall be displayed:

  • For this, select the desired mode by means of the Range mode list box:
    • -All
      entire measurement (default)
    • -Angle
      angle range
    • -Time
      time range
  • In case of Angle or Time selection, the maximum selectable angle or time range will be displayed in the Available area.
  • Enter the range to be displayed in the Selected area.
  • Confirm the settings with OK.
  • The matrix will be loaded. Here, the matrix is the addition of the single values. If you, for instance, select an angle range of 0° up to 90°, a matrix with the total dose for this angle range will be displayed.

For further information on loading matrices, please refer to section General Information on Loading.

Display of the Compare Values

  • Select the Table entry in the list box (1a) in the toolbar.
  • A table with the positions of the reference matrix and the calculated compare values will be displayed.
  • Positions without the calculation of a compare value (not evaluated) are marked with "---".
  • In case of the "Gamma Index" method, the underdosed positions with a Gamma value > 1 are highlighted blue and the overdosed positions with a Gamma value > 1 are highlighted red.

You can transfer these compare values to other software, e.g., spreadsheet programs:

  • Highlight the desired range and select the context menu Copy in order to copy the highlighted compare values to the clipboard. From there, you can paste them to other software.
  • In order to save all compare values as a CSV file (*.csv) or as a text file (*.txt), select the context menu Save All.
    In order to save only a part of the values, highlight the desired range and select the context menu Save Selected.
    A CSV file can be opened directly with a spreadsheet program.

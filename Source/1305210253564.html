Types of Matrices and Patient ID

The names of the matrix types are shown above the matrices A and B.
Types of matrices are:

Measurement:
detector array measurements
MEPHYSTO:
MEPHYSTO mcc data
PTW-Image:
matrices in PTW image format
FIPS-Image:
FIPS Plus data
DoseMap:
matrices of radiation treatment planning system
TIFF-Image:
TIFF images
MATRIX:
simple matrices without information

The patient ID will be added if it is available in the file and if this option is activated in the menu Tools → Options on the Advanced tab (refer to section Advanced Options).

Saving Matrices

Detector array matrices can be saved in PTW mcc format (*.mcc), in TIFF format (*.tif), or in PTW image format (*.dat).

Non-detector array matrices can be saved in TIFF format (*.tif) or in PTW image format (*.dat).

  • To save the matrix A or the matrix B, click the appropriate button or select the menu File → Data Set A/B → Save As.
  • For matrix A or matrix B, the standard Save dialog will be displayed. The proposed file name consists of the current date and time.
  • The matrix will be saved with all the performed image processing.
Hint

In case of dose volumes or lists of matrices, only the currently displayed slice of the dose volume or the currently displayed matrix of the list will be saved.

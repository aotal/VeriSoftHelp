Measurement with Detector Array and Inclinometer

Hint

The following information is not valid when using an inclinometer with the rotation unit. In this case, please refer to section OCTAVIUS 4D Measurement.

The measuring system for this measurement consists of a detector array, an OCTAVIUS phantom, and an inclinometer. The detector array will be inserted into the OCTAVIUS phantom. The inclinometer detects the gantry angle of the radiation device.

For each measurement interval, the gantry angle and the dose values of all measuring chambers of the detector array will be saved.

The measurement result in VeriSoft is a matrix with the summation of the dose values of each measuring chamber.

 

  • Prepare the measurement as described in section Preparing a Measurement.
  • Set the measurement parameters as described in section Setting Measurement Parameters and confirm them with OK.
  • The measuring window will be displayed.
  • Perform the desired measurement and accept the measuring data by clicking the Accept button.
Hint

For further information on performing a measurement, refer to section VeriSoft Measuring Window.

  • The Select an angle range dialog will be displayed.
Select an angle range dialog for gantry angles

In the Select Range pane you determine whether the entire measurement or the measured values of a defined angle or time range shall be displayed:

  • For this, select the desired mode by means of the Range mode list box:
    • -All
      total measurement (default)
    • -Angle
      angle range
    • -Time
      time range
  • In case of Angle or Time selection, the maximum selectable angle or time range will be displayed in the Available area.
  • Enter the range to be displayed in the Selected area.
  • Confirm the settings with OK.
  • The measured matrix will be displayed in the appropriate matrix window as a grayscale image with isodoses. The isocenter is marked with a diagonal cross.
    In this case, the measured matrix is the added sum of the single values. If you, for instance, select an angle range of 0° up to 90°, a matrix with the total dose for this angle range will be displayed.
  • At the same time, the results window will be updated. For instance, the profile representation of the measured matrix will be displayed.

For further information on the display of the matrices, refer to section Measurement with Detector Array.

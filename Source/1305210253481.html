Quick Start (Workflow)

Hint

If both matrices have the same unit, you will obtain the difference very quickly if you perform only the steps in bold print.

  1. Start VeriSoft
  2. Set options:
    menu Tools → Options
  3. Load matrix A
    comparative matrix in matrix window A, normally the data of the radiation treatment planning system
    • -open a matrix or a dose volume:
      Open Data Set A button
    • or
    • -open a list of matrices:
      menu File → Data Set A → Open List
  4. Load matrix B
    reference matrix in matrix window B, normally the measured data
    • -open a stored matrix or a dose volume:
      Open Data Set B button
    • or
    • -measure a matrix with a detector array:
      • -set measurement options:
        menu Tools → Measurement Options
      • -perform measurement:
        Measure Data Set B button
  1. Process the matrices
    (Edit tab or menu Edit → Data Set A/B)
    • -define the isocenter manually:
      Align Manually button
      or
      define the isocenter automatically:
      menu Edit → Align Automatically
    • -define the region of interest:
      ROI button
    • -subtract background:
      Background button
    • -calibrate:
      Calibrate button
    • -normalize:
      Normalize button
    • -rotate or mirror:
      Rotate/Flip/Mirror button
  2. Set graphic options
    • -define isodose levels and colors:
      menu Graphics → Edit Contour Settings
    • -define profile colors and styles:
      menu Graphics → Edit Contour Settings
  3. Show the chamber positions (only if one matrix is a detector array matrix):
    menu View → Overlay Measuring Positions
  1. Display the matrices in the matrix windows
    Switch between graphical (grayscale image, isodose, 3D) and tabular representation
  2. Display the profiles in the results window
    Switch between different representations
  3. Evaluate the matrices in the results window
    Switch between histogram and results
  4. Evaluate the matrices in the compare window
    • -select the compare mode and set the compare parameters:
      Compare tab
    • -switch between graphical and tabular representation
    • -select the compare mode Gamma 3D and perform the comparison of the dose distribution in the volume:
      Compare tab and Calculate Volume Analysis button
    • -overlay the patient anatomy:
      RTView tab
  5. Save the changed matrices:
    Save button
  6. Exit VeriSoft

Numerical View of the Comparison

The numerical view of the comparison contains the following results:

Statistics Area

  • -Number of Voxels
    Total number of voxels in the dose volume
  • -Evaluated Voxels
    Number of voxels, for which a compare value was calculated, i.e., total number minus the number of voxels below a defined threshold. You can set this threshold in the Compare pop-up window of the compare window (refer to section Selecting the Compare Mode and Adjusting the Compare Parameters). The threshold is valid for the total dose volume.
  • -Passed
    Number of Evaluated Voxels that have met the adjusted criteria.
    The percentage ratio to the Evaluated Voxels is indicated in brackets.
  • -Failed
    Number of Evaluated Voxels that have not met the adjusted criteria.
    The percentage ratio to the Evaluated Voxels is indicated in brackets.
  • -Result
    Result according to the evaluation criteria adjusted in the menu Tools → Options on the Compare tab (refer to section Compare Mode Options).

Compare Area

  • -Arithmetic Mean:
    Arithmetic mean of the calculated compare values
  • -Min:
    Minimum calculated compare value with coordinates and value of the reference matrix
  • -Max:
    Maximum calculated compare value with coordinates and value of the reference matrix
  • -Median:
    Median of the calculated compare values

Absolute Difference Area

  • -Arithmetic Mean:
    Arithmetic mean of the dose differences
  • -Min:
    Minimum calculated dose deviation with difference and value of the reference matrix
  • -Max:
    Maximum calculated dose deviation with difference and value of the reference matrix
  • -Median:
    Median of the dose differences

Settings Area

Settings for Gamma 3D compare mode in the menu Tools → Options on the Compare tab (refer to section Compare Mode Options)

Parameters Area

Settings for Gamma 3D compare mode in the Compare pop-up window (refer to section Gamma 2D and Gamma 3D Methods)

 

  • By selecting the context menu Copy, you can copy the numerical view to the clipboard. From the clipboard, you can paste it to other software.
  • By selecting the context menu Save As, you can save the numerical view as a CSV file (*.csv) or as a text file (*.txt).

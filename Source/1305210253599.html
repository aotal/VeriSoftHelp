Creating Batch Files

  • Load a matrix.
  • Start the recording of a batch file by selecting the menu Edit → Start Data Processing Batch Recording.
  • Perform all the desired image processing steps.
  • Finish the recording of a batch file by selecting the menu Edit → Stop Data Processing Batch Recording.
  • The Save Processing Batch dialog will be displayed:
Save Processing Batch dialog
Hint

If during recording of a batch file two matrices are loaded and the image processing affects both matrices (e.g., defining a ROI), two batch files will be recorded and the Save Processing Batch dialog will be displayed for both matrices.

  • Enter the desired file name and confirm with OK.
  • The batch file will be saved.

By clicking the  …  button, you can display a further dialog. This dialog shows the available batch files for matrix A or matrix B. By means of the context menu, you can rename or delete the existing batch files.

The stored batch files are located in the directory '<PTW application files>\VeriSoft\VeriSoft\BatchProcess'.

Options for the Import of DICOM Data

Use DICOM tag ImagePositionPatient for calculation of the isocenter position

When this check box is selected, the isocenter will not be positioned in the center of the loaded matrix. The isocenter information contained in the file will be used instead.

Use appropriate RTPlan object for calculation of the isocenter position

When this check box is selected, the appropriate DICOM RTPlan file corresponding to the selected DICOM RTDose file will be automatically searched and loaded in order to calculate the isocenter position of the matrix.

If exactly one RTPlan file is found and this file can be definitely assigned, it will automatically be loaded. Otherwise, an appropriate message will be displayed.

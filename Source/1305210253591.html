Rotating and Mirroring a Matrix

  • When moving the mouse cursor over the Edit tab, a pop-up window with buttons will be displayed on the left side of the matrix. To rotate the matrix through a given angle (in the mathematical sense) or to mirror the matrix on an axis, click the appropriate button or select the menu Edit → Data Set A/B → Rotate/Flip/Mirror.
  • The Rotate/Flip/Mirror dialog will be displayed:
Rotate/Flip/Mirror dialog
  • To rotate the matrix, select Rotate and an angle of 90°, - 90°, or 180° or enter any angle.
Hint

It is only possible to rotate a detector array through the angles 90°, - 90°, or 180°.

  • To mirror the matrix on the horizontal axis through the point of origin, select Flip.
  • To mirror the matrix on the vertical axis through the point of origin, select Mirror.
  • Confirm the selected function with the OK button.
Hint

The generated matrix is always rectangular. The areas created by the rotation are filled with the value 0.

If you rotate a matrix through the angle x° and subsequently through the angle - x°, you will not obtain the original image.

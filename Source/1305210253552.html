Matrix Window

Matrix window
  1. 1Toolbar
    • aList box to select the matrix representation mode
    • bSlice depth of the displayed matrix
  2. 2Tab to show pop-up windows
  3. 3Graphical or tabular representation of the matrix
  4. 4Position of the profile shown in the results window (only if the matrix is represented graphically)
  5. 5Type of matrix and Patient ID
  6. 6Legend
  1. 7Status bar with
    • aName of the displayed matrix (A or B)
    • bCursor position in the Left-Right direction
    • cCursor position in the Target-Gun direction
    • dOriginal dose value at the cursor position (refer also to section Display of Dose Values in the Status Bar)
    • eMean or interpolated dose value (only with two loaded matrices) (refer also to section Display of Dose Values in the Status Bar)
    • fCorrection factor (only in case of detector array measurements)
    • gPath and name of the displayed matrix If the name is only displayed partially, a tooltip with the complete file name will be displayed when you move the mouse cursor over it.

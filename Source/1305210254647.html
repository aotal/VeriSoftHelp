'Local Percentage Difference' Method and 'Difference in % of Normalization Value of Reference Matrix' Method

Hint

If a detector array matrix and a non-detector array matrix are loaded, the detector array matrix will be used as the reference matrix.

If two detector array matrices or two non-detector array matrices are loaded, matrix B will be used as the reference matrix.

The difference matrix of two detector array matrices is determined by comparison of the dose values of the chambers.

The difference matrix of two non-detector array matrices is determined by comparison of the individual pixels.

In case of a detector array matrix and a non-detector array matrix, VeriSoft will overlay the non-detector array matrix with a grid corresponding to the projected chamber areas and will calculate the dose mean value for each chamber. These dose mean values are compared with the dose values of the detector array matrix.

Calculation formula for the difference matrix with the "Difference in % of Normalization Value of Reference Matrix" method:

R(i,j)

value at the normalization point (i0,j0) of the reference matrix

For these two compare modes, you can, furthermore, define not to perform the calculation if the dose values of the reference matrix are below a defined threshold. For this, select the appropriate check box and enter the threshold value as a percentage of the maximum of the measured slice.
The points excluded from the evaluation will be labeled as not evaluated, displayed with a compare value of 0 in the diagram, and marked with "---" in the table. The points excluded from the evaluation are not considered in the statistics or in the histogram.

Appendix B: Creating PDD Files with Third-Party Devices and Third-Party Software

Measuring Depth Dose Curves

  • Measure depth dose curves with the following settings:
    • -SSD (Source Surface Distance): 85 cm
    • -required field sizes:
      4 cm x 4 cm, 5 cm x 5 cm, 10 cm x 10 cm, 15 cm x 15 cm, and 26 cm x 26 cm
    • -optional field sizes:
      2 cm x 2 cm and 3 cm x 3 cm
    • -number of measuring points:
      equidistant measuring points with a distance of 2 mm
    • -measuring depth:
      0 mm up to 300 mm
  • Save the measured values of all the depth dose curves in a table (e.g., in Microsoft Excel).

Converting to the PTW Format

You have to convert the measured depth dose curves into the PTW format.

  • Start the "TPR/TMR and OCR Table Generator" software from the MEPHYSTO Navigator or by selecting Start → Program Files → PTW → VeriSoft → TableGenerator.
  • Open one of the default PDD files in the directory '<PTW documents>\VeriSoft\Data\PDDData'.
  • Hint

    The default PDD files refer to an SSD of 85 cm or 92 cm. If you want to perform measurements with another SSD, you need to manually edit the "SSD" value in the MCC file using an editor program. It is not possible to automatically adjust the SSD with the TableGenerator software.

  • Save the PDD file with a new name.
  • Close the default PDD file and open your copy.
  • Delete all the columns of the field sizes 0 cm x 0 cm up to 3 cm x 3 cm, 6 cm x 6 cm up to 9 cm x 9 cm, 11 cm x 11 cm up to 14 cm x 14 cm, and 16 cm x 16 cm up to 25 cm x 25 cm with the Del key.
  • Copy the measured values from the table into the appropriate column in TableGenerator. Please ensure that the measuring depths and measured values in your table comply with the values in TableGenerator.
  • Save the file.
  • Exit the software.

Completing the PDD File

  • Start the "TPR/TMR and OCR Table Generator" software from the MEPHYSTO Navigator or by selecting Start → Program Files → PTW → VeriSoft → TableGenerator.
  • Open the mcc file containing the depth dose curves.
  • Normalize the depth dose curves by selecting the menu Scans → Normalize → to Individual Maximum.
  • Save the normalized values.
  • To be able to work with the normalized values, you have to close the file and open it again.
  • Extrapolate values for the field size 0 cm x 0 cm.
    For this, select the menu Scans → Extrapolate to 0 x 0 cm² or click the Extrapolate button.
  • The values for the field size 0 cm x 0 cm will be calculated and inserted as a new column with blue letters.
  • Interpolate values for all missing field sizes in increments of 1 cm.
    For this, select the menu Scans → Interpolate → Columns or click the Columns button.
  • The Interpolate Columns dialog will be displayed:
Interpolate Columns dialog
  • Adjust the values in the Interpolate Columns dialog according to Fig. Interpolate Columns dialog.
  • Confirm the settings with OK.
  • The values of the corresponding field size will be calculated and inserted as new columns with blue letters.
  • Normalize the depth dose curves by selecting menu Scans → Normalize → to Individual Maximum.
  • Save the completed PDD file in the following directory:
    '<PTW documents>\VeriSoft\Data\PDDData'
  • Create a PDD file for each radiation device and each energy. Name the PDD files so that the assignment is unambiguous (e.g., X06Linac1.mcc).
  • Exit the software.

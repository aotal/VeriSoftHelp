Saving Measurements Performed with Detector Array and Planar Rotation Unit or Inclinometer

Saving the Original Measured Values

  • -If the automatic saving of original measured values is enabled in the Measurement Parameters dialog, the original measured values will be saved after the measurement automatically in an xcc file in the selected directory.
  • -If the automatic saving of original measured values is not enabled in the Measurement Parameters dialog, the xcc file will be saved in the same directory as the mcc file (refer to Saving the Added Dose Values).
  • An xcc file contains the measured dose values with the appropriate gantry angles.

Saving the Added Dose Values

  • Save the added dose values in mcc format according to section Saving Matrices.
  • The mcc file only contains one matrix with added dose values.

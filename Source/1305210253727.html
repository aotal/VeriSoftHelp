Error Messages / Troubleshooting

What can I do if…

… the isodoses are not visible?
Check the isodose levels (menu Graphics → Edit Contour Settings).
… the number of Evaluated Dose Points largely differs from the number of Dose Points?
Check the adjusted threshold value in the Compare pop-up window of the Compare window.
… the contrast of a grayscale image is not optimal?
Use the slider in the Contrast pop-up window of the appropriate matrix window to adjust the display range.
… the Zoom mode is not available?
The Zoom mode is only available if two matrices are loaded.
… the Histogram or the Results representation is not available?
  • These evaluations are only possible if two matrices are loaded and both matrices have the same unit.
  • -To change the unit, select the Calibrate function.
  • -To adapt [Gy], [mGy], or [cGy] values automatically during loading, use the Convert to dose option in the Options dialog.
… the profiles cannot be set in the matrices?
Keep in mind that a profile can only be set to a position that is available in both matrices.
… the Undo function is not available?
Undo is not possible for "Defining the region of interest (ROI)" as well as "Rotating through an arbitrary angle (Rotate)".
… the image processing functions are not available?
Check that you are not in Zoom mode. Otherwise, first select the Zoom representation and then the context menu Reset to return to the original view of the matrices.
… a measuring chamber of the detector array becomes defective (e.g., the measuring chamber of Fig. 102)?
Defective measuring chambers can be disabled with the PTW software ArrayCal. In this case, the measuring value for the position in question is calculated by interpolation of the measuring values from the neighboring chambers.
For further information, please refer to the ArrayCal User Manual.

… communication problems occur?
These hints are only valid if the selected detector array is connected via an RS232 connection.
When using long cables (> 20 m) or slow PC interface drivers, communication problems can occur with some PCs with a high baud rate (115200 baud). Sometimes using another RS232 interface (if available) helps. Otherwise, proceed as follows:
  • PTW recommends the communication via a LAN connection (refer to section Settings for Data Transfer).
  • If this is not possible, the use of a USB-RS232 converter may be helpful (for example, LINDY USB-RS232 converter, PTW reference number L242064).
  • Otherwise, you can activate the Reduce RS232 Baudrate option (refer to section Selection of the Measuring Devices). Then, VeriSoft will be able to set only a baud rate < 115200 baud and thus some interval times will no longer be available.
Hint

If the software shows a communication disturbance, the connection possibly cannot be reactivated. In order to remedy the disturbance, a shorter cable link (< 3 m) must be chosen temporarily.

… the display of VeriSoft shows artifacts (e.g., text color and background color are similar or identical)?
In the Control Panel, check the settings for the display:
  • Windows 7:
  • Select Start → Control Panel → Appearance and Personalization → Personalization.
    or
    Select Personalize from the context menu of the desktop background.
  • Select one of the following color schemes. These schemes are recommended by PTW:
    • -Windows 7 - Basic
    • -Windows Classic.
  • Windows 8 / Windows 8.1 / Windows 10:
  • Select Control Panel → Appearance and Personalization → Personalization.
    or
    Select Personalize from the context menu of the desktop background.
  • Select the color scheme "Windows" that is recommended by PTW.
… the text, icons, or VeriSoft are not displayed completely?
In Control Panel, check the settings for the font size:
  • Windows 7:
  • Select Start → Control Panel → Appearance and Personalization → Display.
  • Select the Smaller - 100% (default) option.
  • Windows 8 / Windows 8.1:
  • Select Control Panel → Appearance and Personalization → Display.
  • Select the Smaller - 100% (default) option.
  • Windows 10:
  • Select Control Panel → Appearance and Personalization → Display.
  • Select the 100 % (recommended) size.

Results Window

Results window
  1. 1Toolbar
    • aList box to select the result representation mode
  2. 2Tab to display a pop-up window
  3. 3Representation of the result as profile, histogram, or table
  4. 4Normalization values of the matrices A and B
  5. 5Displayed crosshairs
  6. 6Status bar
    in case of displayed crosshairs with
    • aName of the selected matrix (A or B)
    • bOffset of the profile (not available in case of diagonal profiles) or cursor position on the profile
    • cOffset of the profile (not available in case of diagonal profiles) or cursor position on the profile
    • dMeasured value at the cursor position
    • eCompare value for the selected mode at the cursor position (only available if the reference matrix is selected)
    • fAbsolute dose difference at the cursor position (only available if the reference matrix is selected)

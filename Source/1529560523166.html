Note on Adding Measurements Performed with a Rotation Unit Not Positioned at the Isocenter with Different Energies

Hint

Check that the position of the rotation unit that is not the isocenter is the same for all OCTAVIUS 4D measurements that you add and load.

  • If measurements were performed with irradiation of a rotation unit not positioned at the isocenter with different energies and you want to add these measurements for calculation of a dose volume, in menu Tools → Options on the 4D Dosimetry tab, select the Off-axis 4D dosimetry option in the Non-Standard 4D Dosimetry pane first (refer to section Settings for 4D Dosimetry).
  • In the Select an angle range dialog, you will then see all measurements and one associated PDD file each in the PDD File pane. The Off-axis 4D Dosimetry pane will be displayed in addition:
Select an angle range dialog for calculation of an OCTAVIUS 4D dose volume for irradiation of a rotation unit that is not positioned at the isocenter with different energies
  • In the PDD File pane, for each measurement select the PDD file corresponding to the energy in order to calculate the dose volume.
  • In the Off-axis 4D Dosimetry pane, enter the shift values of the rotation unit for the directions LR, TG, and BT. For further information, refer to section Note on Irradiation Procedures with Rotation Unit Not Positioned at the Isocenter.
  • In the subsequent prompt, confirm the selected PDD files again.
  • VeriSoft will use the PDD file associated with each OCTAVIUS 4D measurement file for calculation and will take the position that is not the isocenter into account.

Hint

For further information on performing measurements with irradiation of a rotation unit not positioned at the isocenter with different energies and on loading these measurements, please refer to the Technical Note D913.200.12 published by PTW.

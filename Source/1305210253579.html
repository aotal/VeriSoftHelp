Defining the Region of Interest in Matrices

  • When moving the mouse cursor over the Edit tab, a pop-up window with buttons will be displayed on the left side of the matrix. To define the region of interest, click the appropriate button or select the menu Edit → Data Set A/B → ROI.
  • If only matrices are loaded, the Set Region of Interest dialog for matrices will be displayed.
    If at least one dose volume is loaded, the instructions given in section Defining the Region of Interest in Dose Volumes apply.
Set Region of Interest dialog for matrices
  • Using the mouse, draw a rectangle to define the area. The dialog will show the coordinates of the upper left-hand corner and of the lower right-hand corner.
  • Or set the corners by modifying the values in the dialog. Adapt the rectangle by clicking the Show Region button.
  • Confirm with the OK button. The area within the rectangle will be enlarged, and all other areas will be removed.
  • You can repeat the above steps to define smaller regions of the image.
Hint

If a ROI is defined for a detector array matrix or an OCTAVIUS 4D matrix, VeriSoft will automatically position the ROI on the borders of the ionization chambers or voxels that are at least partially included in the ROI.

The ROI function always affects both loaded matrices at the same time.

If the darkest or brightest regions of an image are removed, the grayscale images will be recalculated.

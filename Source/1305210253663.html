Saving Dose Volumes and Lists of Matrices

  • To save dose volumes or lists of matrices, click the appropriate button or select menu File → Data Set A/B → Save As.
Hint

For dose volumes or matrix lists, only the currently displayed slice of the dose volume or the currently displayed matrix of the list of matrices will be saved.

Furthermore, you can save OCTAVIUS 4D dose volumes in the DICOM format.

For more information on saving, refer to section Saving and Closing Matrices.

Creating a PDD File for a New SSD

VeriSoft can be used to create a new PDD file for a new SSD (Source Surface Distance) on the basis of a PDD file for a given SSD.

  • For this, the following conditions must be met:
  • -Exactly one PDD file has been selected.
  • -The PDD file contains data for field sizes from 0 cm x 0 cm up to the maximum field size in increments of 1 cm.
  • -The data were measured at an SID (Source Isocenter Distance) of 100 cm.
  • -The field sizes are defined at the isocenter.
  • -All depth-dose curves of the PDD file were measured with the same SSD.
  • -The previous SSD is within (84 … 100) cm.

 

  • Select menu Tools → Options and the 4D Dosimetry tab.
  • In the Generate PDD File pane, click the Select folder button.
  • In the standard Open dialog, select the folder where the desired PDD file is located.
  • All PDD files in the selected folder will be listed.
Generate PDD File pane with loaded PDD files
  • Select the appropriate PDD file by selecting the corresponding check box.
  • Click the Create PPDs for new SSD button.
  • In the Create PPDs for new SSD dialog, enter the new SSD value.

    • Input range: (85 … 100) cm

    The new SSD value can be smaller or greater than the old value.

  • The standard Save dialog will be displayed. The proposed file name consists of the file name of the selected PDD file and the ID "_SSD XXXmm".
  • After successfully saving the data, an appropriate message will be displayed.

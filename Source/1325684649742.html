Loading OCTAVIUS 4D Measurements and Calculating the OCTAVIUS 4D Dose Volume

An OCTAVIUS 4D dose volume is calculated from the data of an OCTAVIUS 4D measurement. In this measurement, the gantry angle and the dose values from all measuring chambers of the detector array are saved for each measurement interval. For further information, please refer to section OCTAVIUS 4D Measurement.

When loading the data, the dose distribution will be calculated for each measurement interval by means of field-size dependent depth dose curves. The dose distributions of all measurement intervals will be added.

For information on creating the PDD files with the required depth dose curves, please refer to section Creating PDD Files.

 

  • To load an OCTAVIUS 4D measurement A or B, click the appropriate button or select menu File → Data Set A/B → Open.
  • Select the appropriate xcc file from the standard Open dialog.
  • The Select an angle range dialog will be displayed:
Select an angle range dialog for the calculation of an OCTAVIUS 4D dose volume

In the Select Range pane, you determine whether the entire measurement or the measured values of a defined angle range shall be used for calculation of the OCTAVIUS 4D dose volume:

  • For this, select the desired mode from the Range mode list box:
    • -All
      entire measurement (default)
    • -Angle
      angle range
  • When you choose the Angle option, the maximum selectable angle range will be displayed in the Available area.
Hint

If you want to calculate the OCTAVIUS 4D dose volume only for a selected angle range, you should only export the data of that angle range in the radiation treatment planning system to enable a meaningful comparison of the data.

  • Enter the range to be calculated in the Selected area.
  • In the PDD File pane, select the PDD file for calculation of the dose volume.
    The directory where the PDD files are located has to be defined via menu Tools → Options on the 4D Dosimetry tab (refer to section Settings for 4D Dosimetry).
  • In the Calculation Limits pane, enter the limits for the size of the dose volume to calculate. The limits depend on the detector array and on the rotation unit used. By clicking the Set to default button, you can reset the limits to the defaults. The default setting of the limits is the maximum range possible for the calculation.
    • -Cylinder Radius [mm]
      Radius of the dose volume to calculate
    • -Cylinder Length [mm]
      Length of the dose volume to calculate
  • Confirm the settings with OK.
  • As the directory with the PDD files contains several files, you have to confirm the selected PDD file again.
    By clearing the appropriate check box in the prompt, you switch off the prompt until VeriSoft is restarted or another PDD file is selected.
  • If the selected PDD file does not contain depth dose curves for all field sizes up to 26 cm x 26 cm (OCTAVIUS Detector 1000 SRS: 11 cm x 11 cm), you must confirm that the depth dose curve of the largest field size is to be used for all missing field sizes.
    By clearing the appropriate check box in the prompt, you switch off the prompt until VeriSoft is restarted or another PDD file is selected.
Query when depth dose curves are missing
  • If the selected PDD file does not contain any information as to whether it was determined for a radiation device with or without FFF mode, the following prompt will be displayed:
Query when FFF information is missing
    • If the PDD file was determined for a radiation device with FFF mode, confirm the query by clicking Yes.
    • The corresponding information (filter = FFF) will be entered in the PDD file and saved.
    • If the PDD file was determined for a radiation device without FFF mode, confirm the query by clicking No.
    • The corresponding information (filter = FF) will be entered in the PDD file and saved.
  • The measured values of the xcc file will be loaded. By means of the PDD file, a dose distribution will be calculated for each measurement interval and the dose distributions of all measurement intervals will be added. If you have selected an angle range, the dose distributions will be calculated and added only for the measurement intervals of this angle range.
  • The slice of the dose volume at a depth of 0 mm will be displayed in the appropriate matrix window.

 

You can display the other slices of the dose volume by means of the slider in the Slice pop-up window.

For further information on loading matrices, please refer to section General Information on Loading.

Workflow with Detector Array and Planar Rotation Unit

The following section describes a typical workflow with a detector array and a planar rotation unit. This measuring system can be used for IMRT (Step & Shoot or Sliding Window), Composite measurement (several superimposed IMRT fields), or IMAT (RapidArc, VMAT) measurements.

For this measurement, the detector array will be inserted into the planar rotation unit and it will be rotated according to the gantry angle, ensuring perpendicular incidence of the radiation beam on the detector array at all times. For each measurement interval, the gantry angle and the dose values of all the measuring chambers of the detector array will be saved. VeriSoft will sum up the dose values of each measuring chamber.

For the analysis, you have to export the appropriate plan from the TPS (radiation treatment planning system).

  1. Set up the OCTAVIUS 4D system and put it into service. For this, refer to the user manual "OCTAVIUS 4D Quickstart". For detailed information, refer also to the user manuals of the OCTAVIUS 4D components.
  2. Load the entire patient plan exported from the TPS into VeriSoft (menu File → Data Set A → Open).
  1. Start the measurement by clicking the appropriate button for matrix B (or selecting the menu File → Data Set B → Measure).
    • The Measurement Parameters dialog will be displayed.
    • Enter the actual temperature and air pressure values as well as a correction factor kUser and a correction factor kEnergy for the beam quality or perform a cross calibration (refer to section Cross Calibration).
    • Click the OK button.
    • The measurement will be started.
    • Deliver the complete plan.
    • Terminate the measurement by clicking the Stop button (or selecting the menu Measurement → Stop).
    • Accept the measured data by clicking the Accept button. The measured data will be saved automatically if the Save measurements automatically in folder check box in the Measurement Parameters dialog is selected.
  1. Evaluate the data in the compare window of VeriSoft.
  1. Instead of performing a measurement, you can load data that was measured before (menu File → Data Set B → Open).

Defining Compare Classes

To be able to perform statistical evaluations of VeriSoft compare results with the PTW software Track-it, a compare class must be assigned to each plan verification.

A compare class is defined by its name and the associated radiation unit.

Classes tab of the Options dialog for the Gamma 2D and Gamma 3D method

On the Classes tab, you will find the following options:

Administration

In the Administration pane, you can create and delete compare classes.

Creating a compare class:

  • In the Class name combo box, enter a name for the compare class (e. g., H&N).
  • In the Radiation unit combo box, enter the associated radiation unit (e. g., Linac1).
  • As you exit the two combo boxes, the new compare class will be displayed in the Class ID list box. The name of the class is composed of the two entries and the "at" sign (@), e. g., H&N@Linac1.

Deleting a compare class:

  • In the Class ID list box, select the compare class that you want to delete, and then click the button to the right of the list box.
  • The compare class will be deleted without any further confirmation prompt.

Deleting a compare class name or a radiation unit:

  • In the Class name combo box or in the Radiation unit combo box, select the entry you want to delete, and then click the button to the right of the list box.
  • The selected entry will be deleted without any further confirmation prompt.
  • If the name or the radiation unit is used in a class ID, it will not automatically be deleted. An appropriate message will be displayed.

Compare Parameters

  • In the Compare mode list box, you can select the compare mode:
  • -Gamma 2D and Gamma 3D
    "Gamma Index" method
  • -Local%Difference
    "Local Percentage Difference" method
  • -DoN (%Diff ref Norm. Value)
    "Difference in % of Normalization Value of Reference Matrix" method

Depending on the selected compare mode, you can set compare parameters according to section Adjusting Compare Parameters.

Result Settings

In the Result Settings pane, you can perform the following settings:

  • -In the first line, you define the maximum compare value for the classification passed.
Hint

For the "Gamma Index" method, this value is fixed and it cannot be changed.

  • -In the second and third lines, you can enter threshold values for the evaluation of the calculated compare values.
    • If the criterion entered in the upper input box is met, the result under Results in the results window is marked with a green dot.
    • If the criterion entered in the lower input box is met, the result under Results in the results window is marked with a yellow dot.
    • If neither criterion is met, the result under Results in the results window is marked with a red dot.

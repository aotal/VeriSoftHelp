Workflow with Detector Array

(without using accessories)

The following section describes a typical workflow with a detector array in case of Step & Shoot. The procedure in case of a Composite (several superimposed IMRT fields) or an IMAT measurement (TomoTherapy, VMAT, RapidArc) is identical. In these cases, however, only one dose matrix will be loaded and compared with a measurement.

  1. Place the detector array on at least 3 cm backscatter material and align it on the treatment table with reference to the position lasers at SSD 99.25 cm (The effective measuring point of the detector array is at a depth of 7.5 mm).
    Observe the Target / Gun orientation indicated on the detector array.
    Add 4.5 cm acrylic plates as build up material (for details refer to the detector array user manual).
  2. Load the entire patient plan exported from the TPS into VeriSoft (menu File → Data Set A → Open List).
    The files must be located in the correct order in one folder.
  1. Start the first measurement by clicking the appropriate button for matrix B (or selecting the menu File → Data Set B → Measure).
    • The Measurement Parameters dialog will be displayed.
    • Enter the actual temperature and air pressure values as well as a correction factor kUser and a correction factor kEnergy for the beam quality or perform a cross calibration (refer to section Cross Calibration).
    • Click the OK button.
    • The measurement will be started.
    • Irradiate the first field at gantry position 0°.
    • When the entire field is irradiated, terminate the measurement by clicking the Stop button (or selecting menu Measurement → Stop).
    • Accept the measured data by clicking the Accept button. The measured data will be saved automatically if the Save measurements automatically in folder check box in the Measurement Parameters dialog is selected.
  1. Evaluate the data in the compare window of VeriSoft.
  2. Select the next field with the scroll function (Slice tab) of Matrix A. Start the next measurement by clicking the appropriate button (or selecting menu File → Data Set B → Measure). Irradiate the second field.
  1. Irradiate and check field by field according to the procedure described above.
  2. For evaluating the entire plan, open the planned data (menu File → Data Set A → Open List) and the measured data (menu File → Data Set B → Open List). Select the Synchronous Data Set Scrolling function in the Tools menu. In both windows, the corresponding matrices to compare will thus be displayed when scrolling.

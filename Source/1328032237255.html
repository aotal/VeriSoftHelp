OCTAVIUS 4D Workflow

The following section describes a typical workflow with an OCTAVIUS 4D system. The OCTAVIUS 4D system can be used for IMRT (Step & Shoot or Sliding Window), Composite (several superimposed IMRT fields), or IMAT (RapidArc, VMAT) measurements.

In case of the OCTAVIUS 4D system, the detector array will be inserted into the rotation unit and it will be rotated according to the gantry angle, ensuring perpendicular incidence of the radiation beam on the detector array at all times. For each measurement interval, the gantry angle and the dose values of all measuring chambers of the detector array will be saved. By means of field-size dependent depth dose curves, VeriSoft will calculate a dose distribution for each measurement interval. The dose distributions of all measurement intervals will be added.

For the analysis, you also have to export a dose volume from the TPS (radiation treatment planning system).

With VeriSoft you can compare the dose distribution in any plane or in the entire volume.

Hint

PTW recommends always exporting RTDose and RTPlan objects from the TPS in order to ensure the correct alignment of the files. If the files are not aligned correctly (displacement or wrong slice selected), the RTPlan object can be used to correct this.

  1. Set up the OCTAVIUS 4D system and put it into service. For this, refer to the user manual "OCTAVIUS 4D Quickstart". For detailed information, refer also to the user manuals of the OCTAVIUS 4D components.
  2. Check the settings for 4D dosimetry in the menu Tools → Options on the 4D Dosimetry tab (refer to section Settings for 4D Dosimetry).
    The adjusted voxel size should meet the voxel size of the TPS.
    A smaller voxel size will extend the calculation time but the accuracy will not be increased.
  3. In the TPS, transfer the patient plan to the phantom of the rotation unit. For this, it is imperative to use the included CT scan.
  4. Export the RTDose and RTPlan objects from your TPS.
  5. Load these TPS data in VeriSoft (menu File → Data Set A → Open).
  6. Start the measurement by clicking the appropriate button for matrix B (or selecting the menu File → Data Set B → Measure).
    • The Measurement Parameters dialog will be displayed.
    • Enter the actual temperature and air pressure values as well as a correction factor kUser and a correction factor kEnergy for the beam quality or perform a cross calibration (refer to section Cross Calibration).
    • Press OK.
    • The measurement will be started.
    • Deliver the complete plan to the rotation unit.
    • Terminate the measurement by clicking the Stop button (or selecting menu Measurement → Stop).
    • Accept the measured data by clicking the Accept button.
    • Select the appropriate PDD file. In order to achieve maximum accuracy you should work with the PDD files of your radiation device. The included PDD files are only examples.
    • The dose volume will be calculated and the slice at a depth of 0 mm will be displayed.
  1. Evaluate the data in the compare window of VeriSoft.
  2. Scroll through the slices of the dose volume by means of the scroll function on the Slice tab. In order to display the corresponding matrices in both matrix windows for the comparison, select the Synchronous Data Set Scrolling function in the Tools menu.
  1. Instead of performing a measurement, you can load data measured before (menu File → Data Set B → Open).

 

To check the PDD file and the alignment of the rotation unit, PTW recommends delivering an open field with 10 cm x 10 cm to the rotation unit and comparing the measured dose volume with the planned dose volume.

  • In the results window, you can check the PDD file in transversal view.
  • In the results window, you can check the alignment of the rotation unit in the coronal view.

Notes on Non-Coplanar Irradiation Procedures

  • In menu Tools → Options, on the 4D Dosimetry tab, select the Non coplanar 4D Dosimetry option.
  • In the TPS, calculate the plan for the phantom used for non-coplanar irradiation.
  • Perform the OCTAVIUS 4D measurement with coplanar irradiation.
  • In the Select an angle range dialog, select the rotation angle of the treatment table from the list box, or enter the correct rotation angle.
  • On the basis of the coplanar measurement data and the rotation angle for the treatment table, VeriSoft will calculate the dose volume for the non-coplanar irradiation.

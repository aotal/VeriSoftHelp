Note on Non-Coplanar Irradiation Procedures

  • If you want to calculate a dose volume for non-coplanar irradiation, in menu Tools → Options on the 4D Dosimetry tab, select the Non coplanar 4D dosimetry option in the Non-Standard 4D Dosimetry pane first (refer to section Settings for 4D Dosimetry).
  • In the Select an angle range dialog, you will then additionally see the Couch Angle combo box in the PDD File pane where you can adjust the rotation angle of the treatment table:
Select an angle range dialog for calculation of an OCTAVIUS 4D dose volume for non-coplanar irradiation
  • Select the rotation angle for the treatment table in the Couch Angle combo box or enter the rotation angle.
    • -If an RTPlan file has been loaded, the rotation angles of the treatment table will be read from the RTPlan file and entered in the combo box. Multiple occurrences of rotation angles are entered only once.
    • -The entered rotation angle must comply with the definition of the standard IEC 61217.
  • On the basis of the coplanar measurement data and the rotation angle for the treatment table, VeriSoft will calculate the dose volume for the non-coplanar irradiation.
    If a rotation angle of 0° is entered, an OCTAVIUS 4D dose volume for coplanar irradiation will be calculated

Hint

For further information on performing measurements with non-coplanar irradiation and on loading these measurements, please refer to the Technical Note D913.200.12 published by PTW.

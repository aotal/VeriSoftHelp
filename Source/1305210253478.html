Intended Use

VeriSoft is a part of the PTW OCTAVIUS system.

Intended Use of the PTW OCTAVIUS System

The PTW OCTAVIUS system is used for dosimetry measurements at a radiotherapy system, e.g. a medical linear accelerator (linac) at electron or photon beams, a particle therapy system or a Cobalt-60 therapy system. The OCTAVIUS system is intended for the measurement of dose distributions and the comparison with the data calculated by the treatment planning system (patient plan verification) and/or for periodic quality assurance (QA) procedures according to the QA plan of the responsible medical physicist (e.g. constancy checks).

Device Description

The OCTAVIUS system is comprised of a two-dimensional detector array with ion chambers uniformly arranged as a matrix and a separate detector interface for data acquisition. The detector array can be placed either in a static cuboid or octagonal phantom or in a rotating phantom. For the measurement of moving radiation sources (e.g. the rotating gantry of a linac), the rotating phantom rotates synchronously with the gantry. An inclinometer, fixed to the gantry, delivers the gantry angle values which are used to align the rotating detector array to the gantry. The OCTAVIUS system is controlled with software for data display and processing.

Indications

The OCTAVIUS system is intended to collect beam data for patient plan verification of a treatment planning system and with regard to machine QA for the following purposes:

  • -IMRT patient plan verification
  • -periodic QA procedures, e.g. constancy checks
  • -beam data analysis according to international dosimetry protocols
  • -measurements after repair or replacement of major components of the radiotherapy system

Contraindications

The OCTAVIUS system is intended for QA purposes and must not be used while a patient is present.
It must not be used in diagnostic radiology.
The resulting measurement data is intended for verification of a patient plan and must not be used to control the radiotherapy system by importing the data into the therapy patient plan.

Intended User

The OCTAVIUS system must be used only by qualified personnel, usually the medical physicist responsible for the radiotherapy system or an authorized person.

General Application Information

VeriSoft is a software for the following purpose in radiotherapy:

  • -IMRT (Intensity Modulated Radiation Therapy) patient plan verification by comparing a verification plan calculated by a treatment planning system (TPS) and a dose measurement.

VeriSoft compares measured dose matrices (e.g., PTW detector array matrices) with corresponding calculated matrices (from treatment planning systems) and verifies the dose in IMRT. Dose distributions and comparisons are evaluated numerically and graphically.

The optional software module "DVH 4D" can be used to calculate dose volume histograms (DVH) on the basis of an OCTAVIUS 4D measurement and to compare these DVHs with the DVHs of the radiation treatment planning system (TPS).

  • For measurements with VeriSoft, the following PTW detector arrays can be used:
  • -OCTAVIUS Detector 729
  • -OCTAVIUS Detector 729 XDR
  • -OCTAVIUS Detector 1000 SRS
  • -OCTAVIUS Detector 1500
  • -OCTAVIUS Detector 1500 XDR
  • -OCTAVIUS Detector 1500 MR
  • -2D-ARRAYseven29
  • -2D-ARRAY XDR

  • Furthermore, you can use the following PTW accessories:
  • -Rotation unit (standard version or modular version) with control unit and inclinometer, i.e., the hardware components of the OCTAVIUS 4D system
  • -Inclinometer

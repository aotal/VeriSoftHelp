Display of Dose Values in the Status Bar

Original Dose Value

  • The original dose value at the cursor position is displayed in the status bar in the field (7d):
  • -detector array matrices:
    original dose value of the chamber
  • -OCTAVIUS 4D matrices:
    original dose value of the voxel
  • -non-detector array matrices:
    The dose value at the cursor position is determined by the so-called 'Average Region', which you can adjust in the menu Tools → Options on the Advanced tab (refer to section Advanced Options).

Mean or Interpolated Dose Value

  • A mean or interpolated dose value will be displayed in the status bar in the field (7e), only if two matrices are loaded and one of the loaded matrices is a detector array matrix. Depending on the types of loaded matrices, this value will be displayed in matrix window A or B.
  • -non-detector array matrix and detector array matrix:
    The chamber positions of the detector array matrix will be projected on the non-detector array matrix. The displayed value is the mean value of the dose values under the projected chamber area.
  • -detector array matrix and OCTAVIUS 4D matrix:
    The chamber positions of the detector array matrix will be projected on the OCTAVIUS 4D matrix. The displayed value is the mean value of the dose values under the projected chamber area.
  • -two matrices of the same type:
    In matrix window A, the interpolated dose value at the position in matrix B will be displayed.

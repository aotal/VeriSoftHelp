Creating Profiles

Creating profiles affects always all the loaded matrices.

Detector Array Matrices:

  • Create a profile by a single mouse click on the matrix.
  • First VeriSoft searches for the nearest chamber center. The profile will be drawn through that point.

OCTAVIUS 4D Matrices:

  • Create a profile by a single mouse click on the matrix.
  • First VeriSoft searches for the nearest voxel center. The profile will be drawn through that point.

Non-Detector Array Matrices:

  • Create a profile by a single mouse click on the matrix.
  • First VeriSoft searches for the nearest pixel. The profile will be drawn through that point.

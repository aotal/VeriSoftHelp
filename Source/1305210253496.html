Define List and Sequence Dialog

Furthermore, you will find the following options for the list of matrices in the Define List and Sequence dialog:

Adding Matrices to the List of Matrices

  • Click the Add File(s) button.
  • In the standard Open dialog, select one or several matrices to add.
  • The selected matrices will be appended at the end of the existing list.

Removing Matrices from the List of Matrices

  • In the list of matrices, select one or several matrices to delete.
  • Click the Remove File(s) button.
  • The selected matrices will be removed from the list.

Emptying the List of Matrices

  • Click the Clear button.
  • The complete list will be deleted.

Changing the Order of the Matrices in the List

  • In the list of matrices, select one or several matrices to move.
  • Move the selected matrices to an upper position in the list by clicking the Move Up button.
  • Move the selected matrices to a lower position in the list by clicking the Move Down button.

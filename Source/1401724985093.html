Merging and Loading OCTAVIUS 4D Measurements

Hint

Only OCTAVIUS 4D measurements can be merged. If measurements with inclinometer but without rotation unit are selected, an appropriate error message will be displayed.

Merging and loading OCTAVIUS 4D measurements is only possible if they have been measured with the same device.

You can only merge two OCTAVIUS 4D measurements where the detector array has been shifted in the direction of the rotation axis.

Merging and loading of OCTAVIUS 4D measurements is not possible in the following cases:

  • -non-coplanar irradiation
  • -irradiation of a rotation unit not positioned at the isocenter
  • -irradiation with different energies

The Merge function at loading of OCTAVIUS 4D measurements (xcc files) is used to calculate an OCTAVIUS 4D dose volume from a database with a higher resolution.

Therefore, two OCTAVIUS 4D measurements have to be available with the detector array shifted 5 mm (OCTAVIUS Detector 1000 SRS 2.5 mm). For the first measurement, the detector array is aligned to the isocenter. For the second measurement, the detector array is shifted in the Target direction. To do so, shift the rotation unit by means of the treatment table.

At the loading, these two OCTAVIUS 4D measurements will be merged. The OCTAVIUS 4D dose volume calculated from the merged measurements can be saved in VeriSoft.

  • The OCTAVIUS 4D measurements will be designated as follows:
  • -measurement #1:
    detector array measurement Gun-Left
    (detector array in isocenter)
  • -measurement #2:
    detector array measurement Target-Left

  • To merge and load an OCTAVIUS 4D measurement A or B, select menu File → Data Set A/B → Merge.
  • The Open Data to Merge dialog (refer to Fig. Open Data to Merge dialog) will be displayed.
  • Open the two measurements by clicking the Open #1 and Open #2 buttons (The Open #3 and Open #4 buttons are disabled).
  • In case of OCTAVIUS 4D measurements, a preview will not be displayed.
  • Confirm the selected measurements with the OK button.
  • The Select an angle range dialog will be displayed. It contains the time and angle range values of the measurement #1.
  • Perform the desired settings as described in section Loading OCTAVIUS 4D Measurements and Calculating the OCTAVIUS 4D Dose Volume.
  • The OCTAVIUS 4D measurements will be merged and the OCTAVIUS 4D dose volume will be calculated.

For further information on merging and loading matrices, please refer to section Merging and Loading Matrices.

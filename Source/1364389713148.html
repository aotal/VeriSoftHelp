Exporting Calculated DVH Curves

The calculated PTW DVH curves are automatically saved to the directory set in menu Tools → Options on the DVH tab under Data Path.

Additionally, you can export the calculated PTW DVH curves in the DICOM format to any directory.

  • To do so, click the appropriate button.
  • In the standard Save dialog, enter a file name and the directory.
  • The PTW DVH curves will be exported.

Creating PDD Files

For the calculation of an OCTAVIUS 4D dose volume from the data of an OCTAVIUS 4D measurement, you need a PDD file for each radiation device and each energy. This PDD file must include depth dose curves for all field sizes from 0 cm x 0 cm to 26 cm x 26 cm in increments of 1 cm. The specified field sizes refer to the isocenter plane.

Hint

If your radiation device only offers field sizes smaller than 26 cm x 26 cm, you should measure depth dose curves up to the maximum possible field size. VeriSoft can also use these PDD files. An appropriate query will be displayed in this case.

Refer to the following sections for a description of creating PDD files from depth dose curves that are measured with PTW devices and PTW software.

For a description of creating PDD files from depth dose curves that are measured with third-party devices and third-party software, refer to Creating PDD Files with Third-Party Devices and Third-Party Software.

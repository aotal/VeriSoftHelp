VeriSoft Projects

VeriSoft allows saving a loaded data set as a project, so plan verifications can be continued at a later point in time or the entire plan verification can be reviewed retrospectively.

  • In this context, the term data set denotes one of the following:
  • -matrix loaded and, if applicable, edited in matrix window A
  • -matrix loaded and, if applicable, edited in matrix window B
  • -matrices loaded and, if applicable, edited in matrix windows A and B
  • The following information is saved in the project:
  • -link to the data used

  • If the data were downloaded from a DICOM server, they will be copied to a subdirectory of the project directory. A link will be set to this copy.

  • Hint

    It is important that you do not move the linked data to another directory. If you do, the link will not work any longer and it will not be possible to open the project.

  • -all changes performed in the Edit menu
  • -in the case of an OCTAVIUS 4D measurement, all settings for calculation of an OCTAVIUS 4D dose volume
  • -all settings for comparison of the matrices in the Options dialog and the Compare tab of the compare window

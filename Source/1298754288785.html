Verifying the License

  • In the PTW License Control Center, click VeriSoft, and then click the Options button.
  • The Options dialog will be displayed:
Options dialog

The Options dialog shows the base version and all of the software options (if any) as well as their license status (also refer to Fig. PTW License Control Center).

Click the base version or an option and then click the Modules button to display the Modules info box:

Modules info box for the base version

In the Modules info box you will see a list of all the modules and their version numbers that are included in the base version or in the option.

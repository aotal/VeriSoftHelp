Display of the Dose Values of the Matrices

By selecting the Table entry in the list box of a matrix window toolbar, you will display the dose values tabularly.

The table contains the positions and the appropriate dose values of the selected matrix.
 

If one of the loaded matrices is a detector array matrix, two tables will be generated for the non-detector array matrix. To display the desired table, click the appropriate tab:

  • -Measurement Resolution tab:
    calculated mean values under the virtual chamber areas
  • -High Resolution tab:
    original dose values
     

You can transfer these dose values to other software, e.g., spreadsheet programs:

  • Highlight the desired range and select the context menu Copy to copy the highlighted dose values to the clipboard. From there you can paste them to other software.
  • In order to save all dose values as a CSV file (*.csv) or as a text file (*.txt), select the context menu Save All.
    In order to save only a part of the values, highlight the desired range and select the context menu Save Selected.
    A CSV file can be opened directly with a spreadsheet program.

Defining the Isocenter Manually

  • When moving the mouse cursor over the Edit tab, a pop-up window with buttons will be displayed on the left side of the matrix. To define the isocenter, click the appropriate button or select the menu Edit → Data Set A/B → Align Manually.
  • The Set Isocenter Position dialog for matrices or for dose volumes will be displayed:
Set Isocenter Position dialog for matrices
Set Isocenter Position dialog for dose volumes
  • Set the isocenter by:
    • -selecting the point to use with a mouse click.
    • -entering the coordinates in the dialog. Check the position by clicking the Show Position button.
Hint

If the new isocenter of a detector array matrix is set by a mouse click, the crosshairs will be positioned automatically to the nearest chamber center. The coordinates of the chamber center will be displayed in the dialog.

If the new isocenter of a detector array matrix is set by entering the coordinates in the dialog, the new isocenter will be positioned to the actually entered position.

If the new isocenter of an OCTAVIUS 4D dose volume is set by a mouse click, the crosshairs will be positioned automatically to the nearest voxel center. The coordinates of the voxel center will be displayed in the dialog. The isocenter will be set to the appropriate voxel center in the whole dose volume.

  • Confirm the new isocenter with the OK button.
  • The new isocenter is marked in the appropriate matrix with a diagonal cross. If two matrices are loaded, the display will be adapted according to the information given in section Adaptation of Two Matrices during Loading.

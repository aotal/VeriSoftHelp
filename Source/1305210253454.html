General Information on Loading

  • Matrices can be loaded in various ways:
  • -Loading individual matrices
  • -Adding and loading matrices
    (only detector array matrices)
  • -Loading dose volumes
  • -Loading lists of matrices
  • -Merging and loading matrices
    (only detector array matrices)
  • -Composing and loading matrices
    (only detector array matrices)

Alternatively, you can insert data from Windows Explorer with Drag & Drop.

Instead of loading matrices you can also measure matrices with a detector array (refer to section Measurement with Detector Array)

If matrices are available in the DICOM format, you can also import them directly from a DICOM directory or a DICOM network server (refer to section DICOM import dialog).

Normally, the data of the radiation treatment planning system are loaded in matrix window A and the measured data are loaded in matrix window B.

When loading data, the existing data will be overwritten. If the existing data are not yet saved, a warning will be displayed and you can save them if required.

The loaded matrix will be displayed in the appropriate matrix window as a grayscale image with isodoses. The point of origin is marked with a diagonal cross. At the same time the results window will be updated. For instance, the profile representation of the loaded matrix will be displayed.

Hint

Matrices whose values have no unit will be represented dimensionless.

Hint

If VeriSoft does not know the point of origin of the matrix to load, VeriSoft will determine the center of the image and will place it there.

Hint

When represented as grayscale images or isodoses, detector array matrices will always be displayed in the size 27 cm x 27 cm (OCTAVIUS Detector 1000 SRS: 11 cm x 11 cm) with a resolution of 0.5 mm. The missing points between two measured values will be determined by interpolation. The points in the border area are extrapolated on the basis of the neighboring measured values. The interpolated values are only used to improve the visualization of dose distribution. They will not be considered for evaluation.

Hint

Because of the finite spatial resolution, the detector array isodoses offer finite accuracy and they are displayed only as an orientation guide. The isodose representation, which is determined by a detector array measurement, is not suitable for a comparison of the measured and calculated data.

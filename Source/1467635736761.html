Saving a Project

  • To save a project, select menu File → Project → Save As.
  • The standard Save dialog will be displayed.
    The proposed directory is '<PTW documents>\VeriSoft\Data\Projects'.
    The proposed file name is composed of the patient name used in the DICOM file and of the date.
  • If required, select another directory or choose another file name.
  • Confirm the entries.
  • The project will be saved in the *.vsp format.

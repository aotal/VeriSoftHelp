Loading Matrices in DICOM Format

  • To load a matrix A or a matrix B, click the appropriate button or select menu File → Data Set A/B → Open.
  • Select the desired file in the standard Open dialog.
  • The DICOM Isocenter Position dialog will be displayed:
DICOM Isocenter Position dialog
  • Define the options for determining the isocenter position:
    • -When the Use DICOM tag… check box is selected, the isocenter will not be positioned in the center of the loaded matrix. The isocenter information contained in the file will be used instead.
    • -When the Use selected RTPlan file… check box is selected, the selected DICOM RTPlan file will be loaded in addition to the DICOM RTDose file in order to calculate the isocenter position of the matrix.
 … 
Button for selecting an RTPlan file
Hint

If the Use selected RTPlan file… option is selected, VeriSoft tries to check whether the RTPlan and RTDose files belong together, but this verification is not always possible.

  • When you select the Autodetect RTPlan file check box, VeriSoft will automatically search for the corresponding RTPlan file in the selected directory.
  • If exactly one RTPlan file is found and this file can be definitely assigned, it will be loaded. Otherwise, an appropriate message will be displayed.
  • By clearing the Show dialog on next DICOM data loading check box, you switch the display of the DICOM Isocenter Position dialog off until restarting VeriSoft, and the last settings will again be used at the next loading of matrices in the DICOM format.
  • Click the Yes button to load the DICOM data and determine the isocenter position using the adjusted options.
  • Click the No button to load the DICOM data without considering the adjusted options.
  • The matrix will be loaded.

For further information on loading matrices, please refer to section General Information on Loading.

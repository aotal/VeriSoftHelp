Image Processing on Lists of Matrices

If a list of matrices is loaded, the image processing functions will be performed only on the actually displayed matrix.

Caution

Scrolling through the processed lists of matrices without saving.

Loss of Data!

If you change a matrix of a list by means of the image processing functions, you have to save these changes immediately. If you scroll through a processed list of matrices, a warning about saving will not be displayed. Performed changes will be lost.

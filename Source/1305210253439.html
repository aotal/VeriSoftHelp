Preparing a Measurement

Before measurements can be performed, set up the program as follows via Tools → Measurement Options (also refer to section Measurement Options):

  • In the Devices in Use pane of the Devices tab, select the measuring devices used:
    • -detector array and associated calibration file (*.cal)
    • -accessories (only for measurements with a rotation unit or inclinometer)
  • In the Connections pane of the Devices tab, configure the data communication settings.
  • On the Measurement tab, set the measuring time.
  • On the Measurement tab, set the interval time.
    After each interval, a new set of measurement values (one value for each channel) is available.
  • On the Advanced tab, adapt the settings for the inclinometer (only for measurements using an inclinometer).
Hint

Please check whether the serial number of the detector array used is the same as the serial number displayed in the status bar. The serial number shown in the status bar is read from the calibration file. VeriSoft will not check the connected hardware.

Activating the Password Protection

  • To activate the password protection, you have to define a password. To do so, click the Set Administration Password button.
  • The Set Password dialog will be displayed:
Set Password dialog
  • Enter the new password and retype it.
  • Confirm the entries with OK.
  • From now on, functions protected with a password can only be accessed after input of the password.
Hint

Upon program start, the password needs to be entered only once.

PTW-Freiburg recommends exiting VeriSoft and restarting it after working with password-protected functions.

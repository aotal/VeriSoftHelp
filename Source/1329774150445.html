Status Bar

  • If the following conditions are met, the values will be displayed in the status bar (6) when the mouse cursor is moved over a failed point:
  • -detector array matrix or OCTAVIUS 4D measurement loaded
  • -compare mode "Gamma 2D" or "Gamma 3D" selected
  • -display of Failed Points enabled
  • The displayed values are:
  • -selected compare mode
  • -cursor position in the Left-Right direction
  • -cursor position in the Target-Gun direction
  • -gamma value at the cursor position
  • -absolute dose difference at the cursor position
  • -relative dose difference at the cursor position

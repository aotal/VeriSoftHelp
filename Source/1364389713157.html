Options in the Graphics Area of the DVH Window

  • To define the dose unit, select Dose Unit and the desired unit Gy or cGy from the context menu.
  • To show or hide the legend or to shift it, select the appropriate option from the context menu:
    • -Legend → Show/Hide:
      The legend will be visible or not visible.
    • -Legend → Upper Right:
      The legend is displayed in the upper right-hand corner.
    • -Legend → Upper Left:
      The legend is displayed in the lower left-hand corner.
  • To change the background color, select DVH Background Color from the context menu.
  • To restore the default settings, select Default from the context menu.
  • Using Copy from the context menu, you can copy the displayed DVH curves to the clipboard. From the clipboard, they can be imported into other programs.
  • With Save As from the context menu, you can save the displayed DVH curves as a graphics file. The available file formats are *.png, *.jpg, *.tif and *.bmp.
  • With Show Cursor from the context menu, you display a crosshairs and a tooltip. By clicking a DVH curve you display a tooltip with the following information:
    • -Name of the corresponding structure
    • -Dose of the dose point next to the cursor position and corresponding volume of the PTW DVH curve
    • -Dose of the dose point next to the cursor position and corresponding volume of the TPS DVH curve

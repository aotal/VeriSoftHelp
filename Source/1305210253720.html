Searching for Existing Connections

The easiest way to define data transfer settings is to search for existing connections with the Search function. For further information on the different communication methods, refer to section LAN Connection and section RS232 Connection.

  • In the Measurement Options dialog, click the Search button.
  • The Devices dialog will be displayed. The Devices dialog includes a list of all connected devices:
    • -When searching for a measuring device, all connected detector interfaces will be displayed (refer to Fig. Devices dialog for searching for a measuring device).
    • -When searching for a measuring device with accessories (control unit or inclinometer), the upper area will show all connected detector interfaces and the lower area will show all connected control units or inclinometers (refer to Fig. Devices dialog for searching for a measuring device and an accessory).
  • If no device can be found, activate the Advanced Search option in the Devices dialog and repeat the search by clicking the Search button.
  • As an alternative, it is possible to manually adjust the data transfer parameters according to section Setting a LAN Connection Manually and section Setting an RS232 Connection Manually.
  • Click the desired detector interface and the accessory (control unit or inclinometer).
  • Click OK to accept the settings.
    With Cancel, you reject the settings.
  • The settings of the selected detector interface and accessory (control unit or inclinometer) will be displayed in the Measurement Options dialog.

For further information on troubleshooting concerning the Search function, refer to section Troubleshooting - LAN Connection.

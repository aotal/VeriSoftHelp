Compare Result as Distribution or Failed Points

The Distribution or Failed Points representation mode is selected by means of the list box (1a) in the toolbar:

  • -In case of Distribution representation, the color gradient depends on the selected compare mode.
    • -"Gamma Index" method:
      color gradient from green (Gamma value = 0) via yellow to red (Gamma value ≥ 1.5)
    • -"Local%Difference" or "DoN (%Diff ref Norm. Value)" method:
      color gradient from blue (negative compare value) via green to red (positive compare value)
  • -In case of Failed Points (only available for "Gamma Index" method) representation, the points with a Gamma value > 1 will be displayed as discrete points.
    A blue point means underdosage and a red point means overdosage.
    When moving the mouse cursor over such a point, the following further information will be displayed as a tooltip:
    • -minimum accepted spatial deviation, for which this point would pass the criterion
    • -Gamma value calculated for this deviation
    • -position of the comparison point
    • -position of the point at the mouse pointer

 

  • If one of the loaded matrices is a detector array matrix, you can project the positions of the chambers to the loaded matrices by selecting the menu View → Overlay Measuring Positions.
  • You can display or hide the legends to the matrices by selecting the menu Graphics → Legend.
  • By means of the list box (1b) in the toolbar of the compare window, you can display further information (refer to section Changing the Visualization in the Compare Window).
  • By selecting the context menu Copy, you can copy the actual representation of the compare result to the clipboard. From the clipboard you can paste it to other software.
  • By selecting the context menu Save As, you can save the actual representation of the compare result as a graphic file. The possible file formats are *.png, *.jpg, *.tif, and *.bmp.

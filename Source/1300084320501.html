Changing the Password or Deactivating the Password Protection

  • To change the password or to deactivate the password protection, click the Set Administration Password button.
  • The Change password dialog will be displayed:
Change Password dialog
  • Type the old password.
  • To change the password, enter the new password and retype it.
  • To delete the password, leave the New Password and Confirm New Password input boxes empty.
  • Confirm the entries with OK.

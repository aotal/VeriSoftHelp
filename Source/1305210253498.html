Merging and Loading Matrices

Hint

This function is only available for detector array matrices. Otherwise, an appropriate message will be displayed.

Merging and loading matrices is only possible if they have been measured with the same device.

The Merge function at loading of matrices is used to generate detector array matrices with higher spatial resolution.

Therefore, 2 or 4 matrices have to be available with the detector array shifted 5 mm (OCTAVIUS Detector 1000 SRS 2.5 mm). In the first matrix, the detector array is aligned to the isocenter. At the following matrices, the detector array is shifted.

At the loading, these matrices will be merged. The merged matrix can be saved in VeriSoft.

Hint

Merging only combines the measuring points of all the loaded measurements without changing the measured values.

  • The matrices will be designated as follows:
  • -matrix #1:
    detector array measurement Gun-Left
    (detector array in isocenter)
  • -matrix #2:
    detector array measurement Target-Left
  • -matrix #3:
    detector array measurement Gun-Right
  • -matrix #4:
    detector array measurement Target-Right
  • Merging of matrices is only possible for the following combinations of matrices:
  • -matrix #1 and #2
  • -matrix #1 and #3 or
  • -matrix #1, #2, #3, and #4

In case of the OCTAVIUS Detector 1500, you can only merge the combination "matrix #1 and #2".

 

  • To merge and load matrix A or matrix B, select menu File → Data Set A/B → Merge.
  • The Open Data to Merge dialog will be displayed:
Open Data to Merge dialog
  1. 1Button to open matrix #1
    (Gun-Left, detector array in isocenter)
  2. 2Button to open matrix #2
    (Target-Left)
  3. 3Button to open matrix #3
    (Gun-Right)
  4. 4Button to open matrix #4
    (Target-Right)

 

  • Open matrix #1 by clicking the Open #1 button.
  • The grayscale image will be displayed in the button. By moving the mouse cursor over the image, a tooltip with path and name of the file will be displayed.
  • Open the matrix #2 or the matrix #3 or the matrices #2, #3, and #4 by clicking the appropriate buttons.
  • The grayscale images will be displayed in the appropriate buttons. By moving the mouse cursor over the images, tooltips with path and name of the appropriate files will be displayed.
  • Confirm the selected matrices with the OK button.
  • The matrices will be merged and loaded.

For further information on loading matrices, please refer to section General Information on Loading.

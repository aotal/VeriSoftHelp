Setting an RS232 Connection Manually

  • In the Measurement Options dialog on the Devices tab, click the appropriate Edit button for the detector interface or the inclinometer.
  • The Edit Connection dialog will be displayed:
Edit Connection dialog for RS232 connection
  • Adjust the parameters required for the connection:
 
Type:
RS232
 
Name:
user-defined name of the connection
 
COM:
used COM port
  • or
  • Search for an existing RS232 connection by means of the Search function according to section Searching for Existing Connections.
    In this case, the Devices dialog will show only the devices connected via the RS232 interface.
  • Check the connection by clicking the Check button.
  • Confirm the settings with OK.
  • The settings of the RS232 connection will be displayed in the Measurement Options dialog.

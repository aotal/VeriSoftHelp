VeriSoft Options

By selecting the menu Tools → Options, you display the Options dialog.

  • By default, the Options dialog comprises 5 tabs for the following topics:
  • -Compare:
    compare mode options
  • -Data:
    options for loading matrices
  • -Advanced:
    advanced options
  • -DICOM:
    settings for the import of DICOM data
  • -4D Dosimetry:
    settings for 4D dosimetry
  • -Classes:
    definition of compare classes for the statistical evaluation in the PTW software Track-it

If a license for the "DVH 4D" option exists, the DVH tab with the "DVH 4D" option settings will also be displayed.

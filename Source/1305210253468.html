OCTAVIUS 4D Measurement

An OCTAVIUS 4D measurement is a measurement with a detector array in a rotation unit.

Types of Matrices

Detector array matrix:
A detector array matrix contains the dose values of a measurement with a detector array.

OCTAVIUS 4D matrix:
An OCTAVIUS 4D matrix is a specific detector array matrix. An OCTAVIUS 4D matrix is thus a slice of an OCTAVIUS 4D dose volume (see below), which is calculated from the dose values of an OCTAVIUS 4D measurement.

Non-detector array matrix:
Non-detector array matrices are all other matrices, e.g., data of the radiation treatment planning system, films, etc.

Dose Volumes

Dose volumes are data sets consisting of several slices. VeriSoft differentiates the following types:

OCTAVIUS 4D dose volume:
An OCTAVIUS 4D dose volume will be calculated on the basis of an OCTAVIUS 4D measurement.

TPS dose volume:
A TPS dose volume contains data exported from the radiation treatment planning system.

Comparative Matrix and Reference Matrix

The comparative matrix is normally the data of the radiation treatment planning system loaded in matrix window A.

The reference matrix is normally the measured data loaded in matrix window B.

Hint

If a detector array matrix and a non-detector array matrix are loaded, the detector array matrix will be used as the reference matrix.

If a detector array matrix and an OCTAVIUS 4D matrix are loaded, the detector array matrix will be used as the reference matrix.

If two detector array matrices, two OCTAVIUS 4D matrices, or two non-detector array matrices are loaded, the matrix B will be used as the reference matrix.

Underdosage

The dose value of the reference matrix is lower than the dose value of the comparative matrix.

Overdosage

The dose value of the reference matrix is higher than the dose value of the comparative matrix.

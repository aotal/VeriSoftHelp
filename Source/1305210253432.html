Error Messages of the Measurement

Message
Cause
Remedy
Missing Zero measurement
A zeroing procedure has not yet been performed.
Perform a zeroing procedure (see section Zeroing).
Device not found
The program cannot connect to the measuring device.
Turn on the device.
Check the cable connections.
Check that the connected device is the device selected in the program.
Calibration file cannot be opened
A calibration file does not exist.
On the Devices tab accessible via the menu Tools → Measurement Options, set the correct path for the calibration file of the connected device (see section Selection of the Measuring Devices).
Invalid calibration file
The calibration file for another device was selected.
On the Devices tab accessible via the menu Tools → Measurement Options, set the correct path for the calibration file of the connected device (see section Selection of the Measuring Devices).
Overload detected
An overload was identified in at least one channel.
Check whether the radiated dose rate exceeds the nominal useful range. If this is not the case, the device must be returned to PTW-Freiburg for repair.
HV Error
A high-voltage error occurred.
Turn the system off and on again. If the problem persists, the device must be returned to PTW-Freiburg for repair.

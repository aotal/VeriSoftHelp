Executing Batch Files

  • Start executing batch files by selecting the appropriate Activate data processing batch check boxes in themenu Tools → Options on the Data tab (refer to section Options for Loading Matrices).
  • When loading a matrix, the Data Processing Batch dialog will be displayed:
Data Processing Batch dialog
  1. 1List of all available batch files
    • aActivated batch file
      The batch file that was used last will always be activated.
  2. 2List of image processing functions of the activated batch file
  • To execute the batch file, click the Apply button.
  • If you do not want to execute the batch file, click the Cancel button.
Hint
  • If executing a batch command is principally not possible because of the command type:
  • -the command will not be displayed in the Data Processing Batch dialog and
  • -the command will not be applied to the data.

By default the Data Processing Batch dialog will be displayed when loading a matrix. If the dialog should not be displayed and the batch file should be executed automatically, you have to clear the Show Dialog before Opening check box. A restart of VeriSoft selects this check box automatically.

By means of the context menu of this dialog, you can rename or delete the existing batch files.

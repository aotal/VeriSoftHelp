Importing Matrices in the DICOM Format

If matrices are available in the DICOM format, they can be imported. You can choose one of these import modes:

  • -DICOM Dir
    Import via any directory with DICOM data,
    e.g., the PTW DICOM directory Storage Directory that is created with the PTW software DicomSCP (refer to the DicomSCP user manual).
    To enable VeriSoft to load DICOM data, the selected directory must contain the DICOMDIR index file. If no DICOMDIR index file exists, an appropriate message will be displayed and you can create a temporary DICOMDIR.
  • -DICOM SCP
    Import of DICOM data directly from a DICOM network server
  • For the DICOM SCP import mode, the network parameters need to be adjusted via menu Tools → Options on the DICOM tab (see section Settings for Importing DICOM Data).
  • Select menu File → Data Set A/B → DICOM Import.
  • The DICOM Import dialog will be displayed (refer to Fig. DICOM Import dialog - DICOM DIR and Fig. DICOM Import dialog - DICOM SCP).

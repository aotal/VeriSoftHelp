Changing the Visualization in the Compare Window

  • The compare result can be represented in the following modes depending on the selected compare mode:
  • -distribution of the compare values
  • -display of failed points (only "Gamma Index" method)
  • -table with compare values
  • -no graphical or tabular representation of the compare result.

The list box (1a) in the toolbar is used to select the representation mode.

 

  • In addition to the compare result the following information can be displayed:
  • -grayscale image of matrix B
  • -isodose areas of matrix B
  • -isodose lines of matrix A
  • -isodose lines of matrix B

The list box (1b) in the toolbar is used to select the additional information. By means of the check boxes, it is possible to combine the elements.

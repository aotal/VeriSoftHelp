Shifting the Profiles

  • Use the Shift key and the cursor keys ­↑, ↓, ← and → to shift the profiles depending on the matrix resolution. If a discrete crosshairs is displayed in the results window, the position of the profile (offset) will be shown there in the status bar.
  • Detector Array Matrices:
    If at least one detector array matrix is loaded, the profile will be shifted by the chamber distance, i.e., 10 mm, in the selected direction (in case of OCTAVIUS Detector 1000 SRS 2.5 mm and in case of OCTAVIUS Detector 1500 5 mm).
  • OCTAVIUS 4D Matrices:
    If at least one OCTAVIUS 4D matrix is loaded, the profile will be shifted by the voxel size in the selected direction.
  • Non-Detector Array Matrices:
    The profile will be shifted in the selected direction.

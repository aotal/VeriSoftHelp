Saving Measuring Data

If needed, the measuring data can be saved in the measuring window.

Measurements with a detector array can be saved in the MEPHYSTO mcc format (*.mcc) or in the CSV format (*.csv).

  • Measurements with a detector array in a rotation unit or with an inclinometer can be saved in the xcc format (*.xcc), in the MEPHYSTO mcc format (*.mcc) or in the CSV format (*.csv).
  • -An xcc file contains the measured dose values with the corresponding gantry angles.
  • -An mcc file or a CSV file only contains the measured dose values of the current measurement.

A CSV file can be opened directly with a spreadsheet program.

  • To save, click the Save button or select menu File → Save.
  • Then enter a file name and the directory in the standard Save As dialog. The proposed file name consists of the current date and time. In the case of measurements with the rotation unit, the file name additionally contains the identification "_4D". You can overwrite existing files or create new files.

Hint

The stored dose values are corrected for all adjusted corrections, e.g., air density correction.

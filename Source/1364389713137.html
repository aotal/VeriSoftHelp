Printing DVH Curves

  • To print the displayed DVH curves, click the appropriate button.
  • First, the Print dialog will be displayed.
Print dialog
  • Enter the administrative data for the printout or the print preview.
Hint

The patient ID and the patient name will be displayed automatically if they are available in the TPS data set.

  • Select the check boxes of the parameters to print or view.
    • -Data origin:
      file names and paths of the loaded OCTAVIUS 4D measurement data and of the TPS data
    • -Custom logo:
      If you wish to insert a special logo, click the Select button.
    • -Confirmation:
      A line for the signature will be added at the end of the page.
  • Select the output format for the print preview from the list box.
  • To define the page layout, click the Page Settings button.
  • To additionally save the print preview or the printout, select the Copy output to folder check box. Click the  …  button and select the desired folder.
  • To display the print preview, click the Preview button.

For further information, please refer to section Printing the Matrices and the Results.

Saving OCTAVIUS 4D Measurements and OCTAVIUS 4D Dose Volumes

Saving the Original Measured Values

After the measurement, the original measured values will be saved automatically in an xcc file with the identifier "_4D" in the directory defined in the Measurement Parameters dialog. The xcc file contains the measured dose values with the corresponding gantry angles.

Exporting the Original Measured Values

Furthermore, the original measured values can be exported as a PTW xcc format file (*.xcc). The xcc file that is saved automatically after the measurement is retained.

  • Select menu File → Data Set A/B → Export → Rotation Unit raw data.
  • The standard Save dialog will be displayed. The proposed file name consists of the current date, time, and the identifier "_4D".

Saving an OCTAVIUS 4D Dose Volume in the DICOM Format

OCTAVIUS 4D dose volumes can be saved in the DICOM format (*.dcm).

  • To save the OCTAVIUS 4D dose volume A or B, click the appropriate button or select menu File → Data Set A/B → Save As.
  • In the standard Save dialog, select the file format DICOM RT and the appropriate directory.
  • The OCTAVIUS 4D dose volume will be saved in the DICOM format.

Saving a Slice of an OCTAVIUS 4D Dose Volume in the DICOM Format

The displayed slice of an OCTAVIUS 4D dose volume can be saved in the DICOM format (*.dcm).

  • Select the slice you want to save.
  • To save the slice of the OCTAVIUS 4D dose volume A or B, click the appropriate button or select menu File → Data Set A/B → Save As.
  • In the standard Save dialog, select the file format DICOM RT - Visible Slice and the appropriate directory.
  • The displayed slice of the OCTAVIUS 4D dose volume will be saved in the DICOM format.

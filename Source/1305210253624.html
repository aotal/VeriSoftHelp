Histogram

If two matrices are loaded, you can display a histogram of the compare result. For this, select the Histogram entry in the list box in the toolbar.

Both matrices must have the same unit. Otherwise, an appropriate message will be displayed and you have to calibrate the matrices first (refer to section Calibration).

  • You can display or hide the legend to the histogram by selecting the menu Graphics → Legend.
  • By selecting the context menu Copy, you can copy the histogram to the clipboard. From the clipboard you can paste it to other software.
  • By selecting the context menu Save As, you can save the histogram as a graphic file. The possible file formats are *.png, *.jpg, *.tif, and *.bmp.

The design of the histogram depends on the selected compare mode (refer to section Selecting the Compare Mode and Adjusting the Compare Parameters).

"Gamma Index" Method:

  1. Dose values that meet the criteria of the Gamma compare mode (passed):
  • Gamma index 0 … ≤ 1:
    • -dark green bars for underdosed values
    • -light green bars for overdosed values
  1. Dose values that do not meet the criteria of the Gamma compare mode (failed):
  • Gamma index 1 < … ≤ 3:
    • -blue bars for underdosed values
    • -red bars for overdosed values
  • Gamma index > 3:
    The dose values will be added and displayed by a blue bar for underdosed values and a red bar for overdosed values.

The gamma increment of the bars is 0.2.

"Local%Difference" or "DoN (%Diff ref Norm. Value)" Method:

The deviations will be displayed by light green bars with an increment of 2 %.

Values with a deviation < -10 % will be added.

Values with a deviation > +10 % will be added.

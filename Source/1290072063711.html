Starting

  • Click the Windows Start button
    → Programs
    → PTW
    → VeriSoft
    → VeriSoft icon.
  • In a licensed version of the software, a window will be displayed with the user name and the version. The main screen (see section Main Screen) will follow.
    In an unlicensed version, the start window with the License button will be displayed first from which the PTW License Control Center can be accessed.

 

  • Alternatively, you can start the software using the MEPHYSTO Navigator.

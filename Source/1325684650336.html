Load Remote Dataset Dialog

Load Remote Dataset dialog

The Load Remote Dataset dialog consists of two panes:

Search Pane

  • In this pane, you can define the criteria for searching patients on the DICOM network server:
  • -Patient ID:
    ID for the identification of the patient
  • -Gender:
    patient's gender
  • -Last Name:
    patient's last name
  • -First Name:
    patient's first name
  • -Date Range:
    period, in which the data have been created
    The following periods are selectable:
    • -All
      no limitation of the date range
    • -Today and Yesterday
    • -Yesterday
    • -Last 7 Days
    • -Last 14 Days
    • -Custom Date Range
      You can define the desired range by means of the list boxes Begin and End.

Patients Pane

In this pane, the patient data sets found will be displayed in tree view.

  • These are the levels of the tree view of the patients on the DICOM network server:
  • -Patient:
    administrative patient data
  • -Study:
    information concerning the treatment
  • -Series:
    information concerning the tests and therapy series as well as the DICOM objects, i.e., matrices, images, and treatment plans

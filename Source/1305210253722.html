Troubleshooting - LAN Connection

The device is connected via a LAN network connection or a LAN direct connection but cannot be found by the Search function.

  1. The Windows firewall prevents the search for the device (broadcast). In this case, an appropriate message will be displayed.
    Unlock the Windows firewall for the network requests of the VeriSoft software:
    • Select the category view.
    • Select Start → Control Panel → System and Security → Windows Firewall.
    • Select Turn Windows Firewall on or off.
    • Switch the Windows firewall off.
Hint

Switching off the Windows firewall is only valid for this session. After restarting the PC and starting a new search with the Search function, you have to switch off the Windows firewall again.

  1. In case of a LAN direct connection, a communication problem occurs.
    Depending on the network interface card of the PC you might connect the device to the PC via a network cable and a CrossOver adapter or via a CrossOver network cable. You can receive a CrossOver adapter from PTW with order number L178090.
  2. In case of a LAN direct connection, the IP addresses of the device and PC are not in the same subnet (range 169.254.x.x).
    You have to configure the network interface card of the PC manually.

For further information, please refer to the Technical Notes "How to Set Up Network Communication" and "How to Set Up a Network Connection" on the VeriSoft installation CD or contact the PTW support.

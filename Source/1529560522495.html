Monitoring of the Bluetooth Connection Status

Hint
  • The status of the Bluetooth connection can be monitored only if the following conditions are met:
  • -VeriSoft ≥ 7.2
  • -OCTAVIUS 4D control unit with firmware version ≥ 2.0

The OCTAVIUS 4D system uses a Bluetooth connection to send the data from the inclinometer to the control unit. An LED on the control unit indicates the status of the Bluetooth connection.

With VeriSoft ≥ 7.2 and an OCTAVIUS 4D control unit with firmware version ≥ 2.0, the status of the Bluetooth connection is sent to VeriSoft:

  • -The status of the Bluetooth connection will be checked before the measurement starts. If a problem occurs, an error message will be displayed and the measurement will not start.
  • -The status of the Bluetooth connection will permanently be checked during the measurement. If a problem occurs, an error message will be displayed and the measurement will be aborted.

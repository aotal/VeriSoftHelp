Status Bar of the Measuring Window

Status bar of the measuring window
  1. 1Status indicator, describes the current action and status
  2. 2Selected detector array
  3. 3Path and name of the loaded calibration file
    (the complete path and name are shown as a tooltip when you move the mouse cursor over this area)
  4. 4Serial number of the selected detector array
  5. 5Current measuring time and measuring value of the selected chamber as well as gantry angle (only in case of a measurement with a rotation unit or inclinometer)

Measured Values in [Gy*]

The calibration factors in the calibration file of a detector array apply to the beam quality 60Co. If the detector arrays are used with beam qualities other than 60Co, the different response would have to be taken into account. Since the corresponding correction factors are not available, the absolute measured value is indicated in the unit [Gy*].

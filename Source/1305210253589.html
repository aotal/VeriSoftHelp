Normalization

  • When moving the mouse cursor over the Edit tab, a pop-up window with buttons will be displayed on the left side of the matrix. To normalize a matrix, click the appropriate button or select the menu Edit → Data Set A/B → Normalize.
  • The Normalize dialog will be displayed:
Normalize dialog

The dialog shows the current normalization value of the selected matrix.

  • Define a new normalization point by clicking a point of the matrix. A cross marker will be displayed and the dialog will show the current position and the corresponding normalization value. You shift the cross marker by clicking another point of the matrix.
  • You can modify the coordinates of the normalization point via the dialog. Click the Apply button to move the cross marker to the new location.
  • Click the Goto Max button to move the normalization point to the position of the maximum value.
  • Irrespective of a selected normalization point, the normalization value can be entered in the dialog. In this case, position values will not be displayed any more.
  • Confirm with the OK button to normalize.

By selecting the Normalize both data sets to position check box, both matrices will be normalized to the same point but to different values.

By selecting the Normalize both data sets to value check box, both matrices will be normalized to the same value.

Hint

Detector array matrix:
If the normalization value of a detector array matrix is defined by a mouse click, the position and the value of the nearest chamber center will be displayed.

OCTAVIUS 4D matrix:
If the normalization value of an OCTAVIUS 4D matrix is defined by a mouse click, the position and the value of the nearest voxel center will be displayed.

Non-detector array matrix:
The value at the normalization point is determined by the so-called 'Average Region' that you can adjust in the menu Tools → Options on the Advanced tab (refer to section Advanced Options).

Hint

The position of the normalization point will not be saved with the matrix data and will be lost when closing the dialog.

Normalizing the data will only affect the isodose image and relative profiles. The original data of the matrix will not be changed.

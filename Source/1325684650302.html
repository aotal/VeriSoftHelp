Selecting and Importing a Matrix from a DICOM Network Server

  • Choose the desired matrix by selecting the appropriate check box at the Study or Series level.
  • Import the selected matrix by clicking the Import button.
  • The DICOM Import dialog will be closed.
  • The matrix will be imported and it will be displayed in the appropriate matrix window as a grayscale image with isodoses. The isocenter is marked with a diagonal cross.
    VeriSoft will proceed as follows:
    • -If you have selected several objects with the same modality, they will be added. If adding is not possible, an error message will be displayed.
    • -If you have selected an RTDose and an RTPlan series and the options Use DICOM tag ImagePositionPatient for calculation of the isocenter position and Use appropriate RTPlan object for calculation of the isocenter position are selected, the isocenter will be calculated accordingly (see also section Options for the Import of DICOM Data).
      If you have selected several RTPlan series, an appropriate message will be displayed and you can select the desired RTPlan series.
  • At the same time, the results window will be updated. For instance, the profile representation of the measured matrix will be displayed.
Hint

If the matrix to import is a dose volume, the display in the preview and in the matrix window is controlled by the slice direction selected in the menu Tools → Options on the DICOM tab (see section Settings for Importing DICOM Data).

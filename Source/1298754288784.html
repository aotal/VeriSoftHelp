PTW License Control Center

  • Click the Windows Start button
    → Program Files
    → PTW
    → License Control Center icon.
  • The PTW License Control Center will be displayed (refer to Fig. PTW License Control Center).
Hint

The License Control Center dialog will show all PTW applications that are currently installed on the PC. When you click a software name and the Register button is disabled, the software either has to be licensed from within the application or it has already been licensed.

PTW License Control Center
  1. 1List of all licensed PTW applications, including the version number and license status.
    Registered for … = correct registration with user information
    Demo expired in … days = remaining days of the demo version trial period
    Demo expired = expired demo version
    Unregistered = unregistered program version
  2. 2Click the button to open the Registration dialog
  3. 3Click the button to view the software options
  4. 4Click the button to close the PTW License Control Center

Explanation of the Toolbar Icons

1. General Buttons

Load a matrix
Measure a matrix with a detector array
Close matrix
Save matrix
Navigate to previous representation in the cycle mode of important representations
Navigate to next representation in the cycle mode of important representations
Export compare result to the Track-it database
Enlarge the graphic window to the size of the program window.
With the context menu Print Preview or Print you can print the enlarged view.
Reset the graphic window to the original size

2. Button with Select List

  • -The displayed icon shows the current selection.
  • -By clicking the button, you select the next button option. The button will show the appropriate icon and the corresponding function will be performed.
  • -A menu showing all options will be displayed in each case by clicking the button. After selecting an option, the button will show the appropriate icon and the corresponding function will be performed.

Selecting the Display Plane

Coronal plane
Sagittal plane
Transversal plane

These buttons will be enabled only if a dose volume is loaded. You adjust the default plane to be displayed in the menu Tools → Options on the DICOM tab (refer to section Settings for Importing DICOM Data).

Defining the Region of Interest in Dose Volumes

  • When moving the mouse cursor over the Edit tab, a pop-up window with buttons will be displayed on the left side of the matrix. To define the region of interest, click the appropriate button or select menu Edit → Data Set A/B → ROI.
  • If at least one dose volume is loaded, the Set Region of Interest dialog for dose volumes will be displayed.
    If only matrices are loaded, the instructions given in section Defining the Region of Interest in Matrices apply.
Set Region of Interest dialog for dose volumes
  • Using the mouse, draw a rectangle to define the area. The dialog will show the coordinates of the upper left-hand corner and of the lower right-hand corner of the rectangle as well as the maximum range of the slice depth.
  • Or set the corners and the range for the slice depth by modifying the values in the dialog. Adapt the rectangle by clicking the Show Region button. Changes of the slice depth range are not indicated.
  • When two dose volumes are loaded and one of them is an OCTAVIUS 4D dose volume, the Adapt to O4D button is active. By clicking the Adapt to O4D button you determine the region of interest whose size corresponds to the calculated OCTAVIUS 4D dose volume.
  • Confirm with the OK button. The region within the rectangle is enlarged while the other regions and, in particular, the regions outside the defined slice depth are removed.
  • You can repeat the above steps to define smaller regions of the image.
Hint

If a ROI is defined for an OCTAVIUS 4D dose volume or detector array matrix or OCTAVIUS 4D matrix, VeriSoft will automatically position the ROI on the borders of the ionization chambers or voxels that are at least partially included in the ROI within the volume.

A dose volume and a matrix are loaded:
The ROI is set for the area defined by the slice depth only when you define the ROI in the dose volume. In all other cases the ROI will only be set in the current slice.

The ROI function always affects both loaded dose volumes / matrices at the same time.

If the darkest or brightest regions of an image are removed, the grayscale images will be recalculated.

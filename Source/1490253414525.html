Further Calculations for 4D Dosimetry

In the Options dialog on the 4D Dosimetry tab, you can choose one of the following options in the Non-Standard 4D Dosimetry pane if additional settings and calculations are necessary for calculation of an OCTAVIUS 4D dose volume:

Non coplanar 4D dosimetry

If the Non coplanar 4D dosimetry option is selected, an OCTAVIUS 4D dose volume can also be calculated for non-coplanar irradiation.

In the Select an angle range dialog (refer to Fig. Select an angle range dialog for the calculation of an OCTAVIUS 4D dose volume or Fig. Select an angle range dialog for the calculation of an OCTAVIUS 4D dose volume), a list box is displayed for this purpose where the rotation angle of the treatment table can be selected or set.

Perform the necessary OCTAVIUS 4D measurement with coplanar irradiation. On the basis of these measurement data, the OCTAVIUS 4D dose volume is calculated for non-coplanar irradiation, taking the entered rotation angle of the treatment table into account.

Off-axis 4D dosimetry

If the Off-axis 4D dosimetry option is selected, an OCTAVIUS 4D dose volume can be calculated even when the rotation unit is not positioned at the isocenter of the radiation unit during irradiation.

In the Select an angle range dialog (refer to Fig. Select an angle range dialog for the calculation of an OCTAVIUS 4D dose volume or Fig. Select an angle range dialog for the calculation of an OCTAVIUS 4D dose volume), three input boxes for shift values in the Left-Right, Target-Gun, and Bottom-Top directions are displayed for this purpose.

Perform the necessary OCTAVIUS 4D measurement. On the basis of these measurement data, the OCTAVIUS 4D dose volume is calculated, taking the entered shift values into account.

4D dosimetry for multiple energies

If the 4D dosimetry for multiple energies is selected, an OCTAVIUS 4D dose volume can be calculated even when the measurements to be added were performed with different energies.

In the Select an angle range dialog (refer to Fig. Select an angle range dialog for the calculation of an OCTAVIUS 4D dose volume or Fig. Select an angle range dialog for the calculation of an OCTAVIUS 4D dose volume), you can select the PDD suitable for the respective measurement in this case.

Perform the necessary OCTAVIUS 4D measurements. On the basis of these measurement data, the OCTAVIUS 4D dose volume is calculated, taking the selected PPDs into account.

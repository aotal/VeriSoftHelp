Appendix D: Settings for Measuring with a 2D-ARRAY XDR

Changing the electrical calibration factor Capacity and the gain factor GainFactor in the configuration file BeamAdjust.xml is described below.

Change these factors only in case of the 2D-ARRAY XDR.

Do not change these factors for the other detector arrays.

Caution

Measuring without adapting the electrical calibration factor Capacity and the gain factor GainFactor in the configuration file BeamAdjust.xml.

Erroneous Measurements!

In the configuration file BeamAdjust.xml, a typical electrical calibration factor and a gain factor are entered. These factors depend on the configuration of the hardware system.

Before measuring check the electrical calibration factor Capacity and the gain factor GainFactor in this configuration file and adapt them if required.

The correct electrical calibration factor Capacity can be found in the calibration certificate of your 2D-ARRAY XDR.

The gain factor GainFactor to enter is 1460E-3.

Caution

Unauthorized modifications of the settings.

Malfunction! Erroneous Measurements!

Only the settings of the parameters described below may be changed in the configuration file BeamAdjust.xml.

Changing any other parameter may cause the malfunction of VeriSoft and erroneous measurements.

The configuration file BeamAdjust.xml is located in the application data directory '<PTW application files>\VeriSoft\VeriSoft'.

  • Make sure VeriSoft is closed.
  • Open the configuration file BeamAdjust.xml using any text editor (e.g., Editor).

The configuration file BeamAdjust.xml has a specific area for each detector array with the appropriate parameters (refer to Fig. Configuration file - Settings area for the 2D-ARRAY XDR).

  • Select the input area for the 2D-ARRAY XDR (1).
  • Enter a gain factor GainFactor (2) of 1460E-3.
  • Enter the electrical calibration factor Capacity (2), which can be found in the calibration certificate of your 2D-ARRAY XDR.
  • Save the modified file with the same file name in the same directory.

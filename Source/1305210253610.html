Profile Representation

If the profile representation is selected, the results window offers the following options:

  • -The profiles for the two matrices are marked in different colors. The colors of the profiles in the matrix windows (refer to Fig. Matrix window, item 4) comply with the colors of the profiles in the results window.
  • You can set the profile colors by selecting menu Graphics → Edit Contour Settings (refer to section Changing Colors and Isodose Levels).
  • -Profiles of detector array matrices are shown as points (not as curves) corresponding to the chamber positions.
    If the compare mode "Gamma Index" is selected and two matrices are loaded, these measuring points are colored according to the compare result:
passed:
green
failed:
blue for underdosage
red for overdosage
  • -By default all profile values are represented relatively to the normalization value of the appropriate matrix. The unit is %.
    The absolute dose values of profiles will be displayed by selecting the Absolute Dose Values check box. The units of the profiles then comply with the units of the matrices.
  • -In case of the diagonals, the zero point corresponds to the vertical axis through the point of origin and the abscissas values correspond to the distance to the point of origin.
  • By selecting the context menu Copy, you can copy the profile representation to the clipboard. From the clipboard you can paste it to other software.
  • By selecting the context menu Save As, you can save the profile representation as a graphic file. The possible file formats are *.png, *.jpg, *.tif and *.bmp.

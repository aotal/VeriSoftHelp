Connecting via a LAN Direct Connection

  • Connect the device directly to the PC via a network cable.
    For the connection, you can use the internal network interface card of the PC or a second network interface card.
  • Switch on the device.
  • As the device cannot find a DHCP server, the device will assign itself automatically an IP address in the range 169.254.x.x via AutoIP.
  • The device and the PC have to be in the same subnet, i.e., their IP addresses have to be in the same range.
    For this, either the network interface card of the PC will also assign itself an IP address in the range 169.254.x.x via AutoIP or you have to configure the IP address of the network interface card manually.

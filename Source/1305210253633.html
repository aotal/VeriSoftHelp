Gamma 2D and Gamma 3D Methods

Hint

If a detector array matrix and a non-detector array matrix are loaded, the detector array matrix will always be used as the reference matrix.

If two detector array matrices or two non-detector array matrices are loaded, matrix B will always be used as the reference matrix.

The "Gamma Index" method is based on the theoretical concept of Daniel A. Low et al. (Med. Phys. 25 (1998) 656-661).

With VeriSoft you have the option to additionally calculate the gamma index by applying the enhanced criteria (second and third pass) elaborated by Tom Depuydt et al. (Radiotherapy and Oncology 62 (2002) 309-319). In specific cases, this leads to fewer failed points. A change will mostly occur in regions of steep dose gradients. This option can be activated and deactivated via menu Tools → Options on the Compare tab (refer to section Compare Mode Options). PTW recommends using this option.

Compare pop-up window with parameters for the "Gamma Index" method

For the "Gamma Index" method, you can define the following calculation parameters:

  • -mm Distance-To-Agreement:
    accepts spatial deviation
Hint

If a dose volume was loaded as the comparative matrix and the Gamma 3D comparison mode was selected, VeriSoft will check the accepted spatial deviation in all spatial directions.

  • -% Dose Difference with ref. to:
    accepted percentage dose deviation
  • -Reference value for the accepted percentage dose deviation:
    • -Local dose:
      dose at the corresponding position of the reference matrix
    • -Max. dose of ...:
      maximum dose of the reference matrix, which depends on the settings in the menu Tools → Options on the Compare tab (refer to section Compare Mode Options)
      • -calculated volume:
        maximum dose of the entire dose volume
      • -measured slice:
        maximum dose of the selected slice
    • -Selected dose:
      dose at an arbitrary point of the reference matrix:
      For this, click the Select button, and then click the desired point in the compare window or enter the desired dose value.
  • -Use increased tolerance:
    You can define a higher tolerance to be used below a defined threshold. For this, you have to select the appropriate check box and enter the higher tolerance and desired threshold value.
  • -Suppress doses below:
    You can determine not to perform the Gamma calculation if the dose values of the reference matrix are below a defined dose threshold. For this, select the appropriate check box and enter the threshold value as a percentage of the defined maximum dose. The maximum dose depends on the setting in the menu Tools → Options on the Compare tab (refer to section Compare Mode Options).
    The points excluded from the evaluation will be labeled as not evaluated, displayed with a gamma value of 0 in the diagram, and marked with "---" in the table. The points excluded from the evaluation are not considered in the statistics or in the histogram.
    PTW recommends selecting this option to eliminate the effects of scattered radiation on the result.

If two dose volumes are loaded and the Gamma 3D compare mode is active, you can initiate the comparison of the dose distribution in the volume with this button (also refer to section Comparison of Dose Distribution in Volume).

Image Processing on Dose Volumes

If a dose volume is loaded, the image processing functions will be performed simultaneously on all slices. When scrolling, the changes will be retained and can be saved at any time.

Hint

VeriSoft will save the currently displayed slice of the dose volume and not the complete dose volume.
VeriSoft will save the data of TPS dose volumes as TIFF files and data of OCTAVIUS 4D matrices as PTW mcc files.
